18. How to get the Network ID of the IP Address?​


Answer:

The network ID is found by logically ANDing the binary form of the IP address with the binary form of the subnet mask for the network. For example, if a host has an IP address of 172.16. 8.55 on a network with a subnet mask of 255.255. 0.0 (the default subnet mask), the network ID of the host is 172.16

27. it refers to the central point that the author id trying to communicate in his/her work​


The central point that the author id trying to communicate in his/her work is main idea. ​The main idea is the central, or most important, idea in a paragraph or passage. It states the purpose and sets the direction of the paragraph or passage.

The key thought or concept that the author wishes to convey to the readers about the subject is the main idea of a paragraph. Consequently, the topic phrase in a paragraph is where the primary concept is directly communicated. It provides the main idea of the paragraph's subject and is supported by the specifics in the sentences that follow. The thesis statement of a multi-paragraph piece states the core idea, which is subsequently backed up by other supporting details. One of the most crucial reading abilities to learn, along with ideas like drawing inferences, determining the author's aim, or comprehending vocabulary items in context, is determining the primary idea of a paragraph or longer passage of text. Consider the primary idea as a succinct but comprehensive summary. It provides a general overview of everything discussed in the paragraph but leaves out the details. Later lines or paragraphs will provide those specifics, which provide context and nuance. The core idea will need these details to sustain its case. Sometimes the main point is not communicated explicitly. A key notion that is indicated or assumed is more difficult to pinpoint. Through the use of other terms in the text, it can be inferred. An implied main idea can be found in several ways.

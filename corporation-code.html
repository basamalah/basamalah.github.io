27. the zip code used by the philippine postal corporation (philPost) to distribute males and packages consist of 3 digits how many zip codes are possible if they are no digit restriction?​


Answer:

1,000

Step-by-step explanation:

Since the zip code consists of three digits and there are no digit restrictions, each digit can take on any value from 0 to 9. Therefore, there are 10 possible choices for each of the three digits, which gives us a total of:

10 x 10 x 10 = 1,000

So, there are 1,000 possible zip codes if there are no digit restrictions.

18. If charmaine can run 3km in an hour how far can she run in 45 minutes


Velocity

Based on the case from the question.

If Charmaine run 3 km in an hour, Charmaine run with velocity of 3 km/hour. Whereas, if Charmaine run with constant velocity in 45 minutes, Charmaine can run 2,25 km.

Let's break it down

We have known that the distance an object goes in one unit of time is known as its velocity. The standard unit of velocity measurement is M/second. The units of velocity include kilometers per hour, minutes, seconds, and so on.

The velocity can be measured by the following formula

V = [tex]\frac{S}{T}[/tex]

where

V is velocity in km/hours, S is distance in kilometer/meter, and T is time (in second/minutes/hours)

So, the running distance of Charmaine can be measured by the following ways:

First, we find the velocity of Charmaine with distance 3 km (V) in 1 hour (T)

V = [tex]\frac{S}{T}[/tex]

  = [tex]\frac{3 km}{1 hour}[/tex]

  = 3 km/hour

Then, we consider the the running velocity of Charmaine is constant, so we can find the running distant (S) of Charmain in 45 minutes (T)

V = [tex]\frac{S}{T}[/tex]

S = V x T

V = 3 km/hour

T = 45 minutes = ¾ hour

S = ?

S = 3 x ¾ = 9/4  = 2.25 km.

So, Charmaine run 2,25 km in 45 minutes with velocity of 3 km/hour.


Simply, the velocity can be measured by dividing the distance to the time

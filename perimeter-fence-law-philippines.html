15ft.

2W = L+5

P= 2L + 2W

P= 2L + (L+5)

80 = 2L + L + 5

80 = 3L + 5

80-5 = 3L

75 = 3L

divide both sides by 3 to get L (length).

L = 25

finding the width:

2W = L+5

2W = 25+5

2W = 30

divide both sides by 2 to get W (width).

W = 15

Let:

A = area

L = length = x

W = width

We need to find the value of W in terms of x, we can use the formula for the area

A = LW = xW

400 = xW

W = 400/x

From P = 2L + 2W

P(x) = 2x + 2(400/x)

P(x) = 2x + (800/x)

P(10) = 2(10) + (800/10)

P(10) = 100 meters

For x = 50

P(50) = 2(50) + (800/50)

P(50) = 116 meters

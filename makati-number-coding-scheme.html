Step-by-step explanation:

BCD Code - BCD stands for binary coded decimal. It is a 4-bit code. It means that each decimal digit is represented by 4 binary digits. It was used by early computers.

EBCDIC Code - EBCDIC stands for extended binary coded decimal interchange code. It is an 8-bit code. It is normally used in mainframe computers. It can represent 256 characters.

ASCII - ASCII stands for American standard code for information interchange. It was published in 1968 by ANSI (American National Standard Institute). It is the most widely used coding scheme for personal computers. The 7-bit code can represent 128 characters. It is not enough to represent some graphical characters displayed on computer screens.

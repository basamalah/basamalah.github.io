14. the ratio to males cows in the farm is 5:1.how many male and how many female cows are there in a farm of 144 cows?( use partative proportion )​


Answer:

To use the partitive proportion method to solve this problem, we can set up the following equation:

5 males : 1 female = x males : 144 cows

Where x is the number of male cows in the farm. To find x, we can cross-multiply and simplify:

5 * 144 cows = 1 * x males

720 = x males

So, there are 720 male cows and 144 - 720 = 144 - 720 = 144 female cows in the farm.

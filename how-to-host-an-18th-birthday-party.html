Let:
10th birthday be the first term.
=> A1 = 300
18th birthday be the 9th term which is asked.
=> A9 = ?
200$ be the common difference
=> d = 200

An = A1 + (n-1)d
A9 = 300 + (9-1)200
A9 = 300 + (8)200
A9 = 300 + 1,600
A9 = 1,900

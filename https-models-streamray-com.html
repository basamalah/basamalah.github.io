3. The three basic communication models: Linear Model, Transactional Model, and Interactive Model. Below is an illustration that shows the descriptions as well as the pros and coms of each model.​


Question:

Differentiate the 3 models of communication

1. Linear Model

2. Interactive Model

3. Transactional Model

Explanation:

Communication

Communication is the act of sending information from one location, person, or organization to another.Usually, communication includes a sender, a message, and a recipient.

Models of Communication

Communication models are theoretical or conceptual frameworks that explain how people communicate. It also symbolizes the complete communication process between the sender and the receiver.

3 Models of Communication

1. Linear Model of Communication

The linear communication model is a one-way interaction in which there is no feedback. The linear communication model is the most common, and the transactional model is built on top of it. Without receiving feedback, the sender connects with the receiver. It also depicts a one-way communication process.

2. Interactive Model of Communication

The interactive model of communication refers to a two-way communication system that includes feedback. However, because feedback is not instant, it is delayed and indirect. When receivers do not respond to senders, communication might become linear. The interactive communication model implies both mediated and internet-based communication.

3. Transactional Model of Communication

The transitional communication model claims to be a two-way communication process with immediate feedback. Simultaneous feedback is a critical component of transitional communication models. Furthermore, the feedback is immediate and direct. The receiver is compelled to respond.

Allan's share:  2x

x + 2x = 960

3x = 960

3x/3 = 960/3

x = 320

Substitute 320 to x:

Ben's share: x = Php 320
Allan;s share: 2x = 2(320) = Php 640

ANSWER:  Ben's share is Php 640.

320 + 640 = 960

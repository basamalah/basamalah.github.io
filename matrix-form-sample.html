1. Explain how a sample is related to or different from the terms analyte and matrix


Answer:

sample: the object of the analytical procedure (for example: a blood sample); analyte: the substance that is of interest in the analysis (for example: amount of hemoglobin in blood); matrix: the constituents, apart from the analyte, of the given sample (for example: all the constituents of blood except hemoglobin);

6. A binary classifier was evaluated using a set of 1,000 test samples in which 50% of all the samples are negative. It was found that the classifier has 60% sensitivity and 70% accuracy. Write the confusion matrix​


A confusion matrix is a table that is used to evaluate the performance of a binary classifier. The rows of the matrix represent the predicted classes, while the columns represent the true classes.

In this case, the binary classifier is being tested on a set of 1,000 samples, 50% of which are negative. We are told that the classifier has 60% sensitivity, which means that it correctly identifies 60% of the negative samples. We are also told that the classifier has 70% accuracy, which means that it correctly classifies 70% of the total number of samples.

Based on this information, we can construct the following confusion matrix:

                                 | Positive | Negative

                                 |----------|---------

                          True    |  350     |  150

                          False   |  30      |  470

This confusion matrix indicates that the classifier correctly classified 350 out of the 500 positive samples (70% accuracy), and correctly identified 470 out of the 500 negative samples (60% sensitivity).

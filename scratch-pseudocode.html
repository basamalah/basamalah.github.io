20. 1. Differentiate Algorithms, Pseudocodes and Flowchart.


Algorithms: An algorithm is a set of instructions for solving a problem. It is a step-by-step process that outlines how a task should be completed. Algorithms can be written in any language, such as English, pseudo-code, or C++.

Pseudocode: Pseudocode is a mixture of programming language and English. It is a way of writing an algorithm that is readable and understandable by humans. It is not an exact representation of the code that will be used in the final program, but it is a useful tool for planning and understanding the logic of a program.

Flowchart: A flowchart is a graphical representation of an algorithm. It is a diagram of boxes, arrows, and other symbols that describe the steps in a process. Flowcharts are used to clearly and logically communicate the steps in a process.

SET THEORYAnswers with Step-by-step explanation:

1.) M= {s, a, f, e), N = {s, e, a}, then N ⊆ M

Answer: SUBSET

Explanation: A subset is a kind of set where all of its elements also belong in the universal set or another set. In the given above, notice that all of the elements in Set N can be seen in Set M, therefore that confirms the mathematical statement that Set N is a subset of Set M.

2.) U = {x/x are numbers between 1 and 9), D = {2, 4, 6) then D' = {1, 3, 5, 7, 8, 9}

Answer: COMPLEMENT

Explanation: The complement of a set can be denoted by an apostrophe. This operation of set gets all the elements in the universal set that is not included in a particular set. In the given above, since the universal set consists of all numbers between 1 and 9 and it is also given that Set D is composed of 2, 4 and 6, hence, the complement of Set D must be composed of 1, 3, 5, 7, 8 and 9.

3.) H=(2, 4, 6, 8) and K = (3, 6.9) then H∩K= {6}

Answer: INTERSECTION

Explanation: The intersection of sets is composed of the elements by which these sets have in common. In the given above, the common element for Set H and Set K is only 6. Therefore, the intersection of Set H and Set K is 6.

4.) A= {x/x is a vowel letter in an English alphabet}, B= {x/x is a consonant letter in an English alphabet}, AUB = {x/x are letters in English alphabet}

Answer: UNION

Firstly, question written correctly is

Evaluate the following limits:

[tex]\lim_{y \to -2} (\cfrac{4-(3y)^2-y^3}{6-y-y^2})[/tex]


Then we use simplest rule, to evaluate the limit of any continuous function as x approaches a given value c, you should simply evaluate the function at that value.

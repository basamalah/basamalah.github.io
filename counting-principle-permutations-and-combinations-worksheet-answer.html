6. fundamental counting principle permutations journalplss pasagot ng maayos: (​


Answer:

The Fundamental Counting Principle and permutations are both concepts in combinatorics, which is the study of counting and arranging objects.

The Fundamental Counting Principle is a rule that states that if there are m ways to perform one task and n ways to perform another task, then there are m x n ways to perform both tasks together. This principle is often used to count the number of outcomes in a multi-step process. For example, if you have 3 shirts and 2 pairs of pants, the Fundamental Counting Principle tells us that there are 3 x 2 = 6 possible outfits you can create by choosing one shirt and one pair of pants.

Permutations, on the other hand, are a type of counting problem that involve arranging objects in a specific order. The number of permutations of n objects taken r at a time is given by the formula:

P(n,r) = n! / (n-r)!

where n! represents the factorial of n, or the product of all positive integers from 1 to n. For example, if you have 5 books and you want to arrange them in a specific order on a shelf, there are 5! = 120 possible ways to do so.

Journals, in the context of combinatorics, can refer to a record of the counting methods used to solve a particular problem, as well as any insights or observations made during the process. Keeping a journal can help in understanding the problem-solving process and can also be useful for future reference.

14. Why do you think that in making plate numbers count for permutation and not combination?


Answer:

Sometimes it makes sense to count the number of ways for an event to occur by looking at each possible outcome. However, when there are a large number of outcomes this method quickly becomes inefficient. If someone asked you how many possible regular license plates there are for the state of California, it would not be feasible to count each and every one. Instead, you would need to use the fact that on the typical California license plate there are four numbers and three letters. Using this information, about how many license plates could there be?

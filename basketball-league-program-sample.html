19. In a basketball league consisting of 12 teams, each team plays each of the other teams exactly twice.How many league games will be played?


Let's first take into consideration how many teams are going to play in the league.  

There are cases where 10 teams play each other twice or 8 teams play each other once. These types of games are usually played in games like badminton, tennis, or even table tennis.  

So we need to take into consideration if there are n team in a league and each team plays each other twice. First we have 12 teams; each game will require 2 teams to play each other. If each team will play each other 1 time, then this would be called a Round-Robin. A round-robin allows each team to play each of the other team once, however; there are cases where each team will play each other 2 times and this is called a Double Round-Robin.  

First we need to calculate how many games will be played in a single round-robin or 12 teams play each other once and find how many games will be played.

12 teams will require 11 rounds to make a complete single game match for all. Since each game will require 2 teams to compete, we can then take the number of rounds and divide that by 2.

12 x 11 = 132/2 = 66 Games

A total of 66 games will be played for a SINGLE ROUND-ROBIN. Now that we have the total number of games that can be played in a single round-robin, we can then take the number of games and just multiply the number of games by 2 since each team will play each other twice.

Single Round-Robin = 66 games  

Double Round-Robin = 66 games x 2 = 132 games total.

12. There are no more than 200 books in a cabinet. If 2 books, 3 books, 4 books, or 6 books are taken out each time, only 1 book remains. Every time 11 books are taken out, there are no books left. How many books are there?


Answer:

121

Step-by-step explanation:

Required Answer:

No. of books at the beginning

Solution:

[tex]\textsf{Assume the cabinet has n books.}[/tex]

[tex]\textsf{Since every time 11 books are taken out, there are no books left, }\sf 11 \: \big\vert \: n[/tex]

[tex]\sf When \: we \: divide \: n \: by \: 2, 3, 4, or \: 6, \: there \: is \: a \: remainder \: of \: 1, \: so[/tex]

[tex]\sf GCF(2,3,4,6) \: \big\vert \: n-1[/tex]

[tex]\implies \sf 12 \: \big\vert \: n - 1[/tex]

[tex]\sf Hence, \: let \begin{cases} \sf n =12x + 1 \\\sf n = 11y \end{cases} \sf x,y \in \mathbb{Z}[/tex]

[tex]\sf It \: follows, \: 11y-12x = 1.[/tex]

[tex]\sf It \: is \: obvious \: that, y_{0} = -1; \: x_0=-1[/tex]

[tex]\sf This \: implies \begin{cases} \sf y = -1+12k\\\sf x= -1 + 11k\end{cases} k\in \mathbb{Z}^{+}[/tex]

[tex]\sf \therefore 1 \leq x \leq 16 \: and \: 1 \leq y \leq 18,[/tex]

[tex]\sf so \: that \begin{cases} \sf 1 \leq -1 +12k \leq 18\\\sf 1 \leq -1+11k \leq 16\end{cases} \implies k = 1.[/tex]

[tex]\sf From \: k = 1 \begin{cases} \sf y = -1+12(1)=11\\\sf x = -1 + 11(1) = 10\end{cases}[/tex]

[tex]\sf \therefore n = 11(11) = 12(10) + 1 = \boxed{\sf 121}.[/tex]

Answer:

103,680 ways

Step-by-step explanation:

The number of arrangements per book is determined as,

Math = 5!

Science = 4!

English = 3!

Number of ways that 3 different books of the same subject be grouped together = 3!

Using Fundamental Counting Principle, we have

5!4!3!3!=(120)(24)(6)(6)=103,680

2. For items 2-3. In OH, the diameter is 52 units long,HN = 13x and ZHAN = 30°.2. What is the value of x?A.1/2B.1C.2D.4​


Answer:

C

Step-by-step explanation:

We can start by using the fact that OH has a diameter of 52 units. Since H is the midpoint of OH, the distance from O to H is half of the diameter, which is 26 units.

Next, we can use the fact that HN = 13x. We know that H is the midpoint of OH, so HN is half of OH. Therefore, we can write:

HN = OH / 2

13x = 52 / 2

13x = 26

Dividing both sides by 13, we get:

x = 2

So the value of x is 2.

5. Two persons on opposite side banks of a canal pull a vessel moving parallel to the banks by means of two horizontal ropes. The tensions in the ropes are 2000N and 2400 N while the angle between them is 70 deg. Determine the resultant pull on the vessel and the angle between each of the ropes and the sides of the canal.​


Answer:

a. The resultant pull is 3611.55 N

b. The angle between each rope is -86.36°

Explanation:

Here, we are to solve for the resultant tension or pulling force on a vessel.

To analyze the problem, I have provided a free body diagram below.

For the formula, we will be using:

[tex]F=\sqrt{(\Sigma Fx)^2+(\Sigma Fy)^2 }[/tex]

[tex]\theta =tan^{-1}(\frac{\Sigma Fy}{\Sigma Fx} )[/tex]

For the given information

F1 = 2000 N

F2 = 2400 N

θ = 70°

Solving the problem

1. First, let us solve for the horizontal components of the given tension forces, we have:

F1x = 2000 cos 55 = 1147.15 N

F2x = - 2400 cos 55 = - 1376.58 N (left side - negative)

Then, we get the summation of them, we have:

∑Fx = F1x - F2x

∑Fx = 1147.15 N - 1376.58 N = -229.43 N

2. Now, let us solve for the vertical component of each tension forces, we have:

F1y = 2000 sin 55 = 1638.30 N

F2y = 2400 sin 55 = 1965.96 N

Then, let us also solve for the summation, we have:

∑Fy = F1y + F2y

∑Fy = 1638.30 N + 1965.96 N = 3604.26 N

3. Now, let us solve for the resultant force using the formula above, we have:

[tex]F=\sqrt{(\Sigma Fx)^2+(\Sigma Fy)^2 }[/tex]

[tex]F=\sqrt{(-229.43 )^2+(3604.26 )^2 }[/tex]

F = 3611.55 N

4. Then let us solve for the angle between the resultant forces, using the formula above, we have:

[tex]\theta =tan^{-1}(\frac{\Sigma Fy}{\Sigma Fx} )[/tex]

[tex]\theta =tan^{-1}(\frac{3604.26 }{-229.43 } )[/tex]

θ = - 86.36°

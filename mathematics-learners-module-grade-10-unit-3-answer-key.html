Mathematics 9 Quarter 1-Module 2: Solving Quadratic Equations by Extracting Square Roots

(Week 1 Learning Code – M9AL-Ia-2.1)

Attached is the guide for you to answer the problems in the module.

Solving Quadratic Equations will enable you to find the values of the variable in quadratic equations using the four different methods.

There are four lessons in this module on solving quadratic equations. Each time you complete a section of this program, you will be able to demonstrate the fundamental ideas of solving quadratic equations via (a) extracting square roots, (b) factoring, (c) square-rooting, and (d) applying the quadratic formula.

Lesson 2a. Solving Quadratic Equations by Extracting Square RootsLesson 2b. Solving Quadratic Equations by FactoringLesson 2c. Solving Quadratic Equations by Completing the SquareLesson 2d. Solving Quadratic Equations by Using the Quadratic Formula

Additionally, by employing real-world examples, you will be able to create and resolve difficulties. These four techniques are used to solve quadratic equations.

To use factoring to solve a quadratic equation:

Use standard form to transform the equation so that one side is zero.Factor the side that is not zero.Put each variable at 0 (Remember: a product of factors is zero if and only if one or more of the factors is zero).Work out each ensuing equation.

How can square roots be extracted to solve quadratic equations

Taking square roots outPut the quadratic equation in standard form as a first step.Factor the quadratic expression in step two.Step 3: Use the zero-product property and set a value of 0 for each variable factor.Solve the resulting linear equations in step four.

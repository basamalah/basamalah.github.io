24. There are 120 envelopes in a box. Cheryl uses 24 envelopes and then 52 envelopes. What percent of the original number of envelopes does she have left?


Answer:

Cheryl has 36.67% of the original number of envelopes left.

Step-by-step explanation:

Cheryl starts with 120 envelopes. She uses 24 envelopes, so she has 120 - 24 = 96 envelopes left.

She then uses an additional 52 envelopes, so she now has 96 - 52 = 44 envelopes left.

To find what percentage of the original number of envelopes Cheryl has left, we can use the following formula:

percentage = (remaining number of envelopes / original number of envelopes) x 100%

Plugging in the numbers we have, we get:

percentage = (44 / 120) x 100%

percentage = 0.3667 x 100%

percentage = 36.67%

Therefore, Cheryl has 36.67% of the original number of envelopes left.

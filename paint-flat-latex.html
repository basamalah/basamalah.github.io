1. n = 10, x̄ = 28, s = 4, 90% confidence

Using a t-distribution with 9 degrees of freedom and a 90% confidence level, the t-value is 1.833.

The standard error of the mean is s/√n = 4/√10 = 1.264.

Therefore, the confidence interval is 28 ± (1.833)*(1.264) = (25.31, 30.69).

2. n = 16, x̄ = 50, s = 4.2, 95% confidence

Using a t-distribution with 15 degrees of freedom and a 95% confidence level, the t-value is 2.131.

The standard error of the mean is s/√n = 4.2/√16 = 1.05.

Therefore, the confidence interval is 50 ± (2.131)*(1.05) = (47.67, 52.33).

3. n = 20, x̄ = 68.2, s = 2.5, 90% confidence

Using a t-distribution with 19 degrees of freedom and a 90% confidence level, the t-value is 1.729.

The standard error of the mean is s/√n = 2.5/√20 = 0.559.

Therefore, the confidence interval is 68.2 ± (1.729)*(0.559) = (67.28, 69.12).

4. n = 23, x̄ = 80.6, s = 3.2, 95% confidence

Using a t-distribution with 22 degrees of freedom and a 95% confidence level, the t-value is 2.074.

The standard error of the mean is s/√n = 3.2/√23 = 0.67.

Therefore, the confidence interval is 80.6 ± (2.074)*(0.67) = (78.83, 82.37).

5. n = 25, x̄ = 92.8, s = 2.6, 99% confidence

Using a t-distribution with 24 degrees of freedom and a 99% confidence level, the t-value is 2.797.

The standard error of the mean is s/√n = 2.6/√25 = 0.52.

Therefore, the confidence interval is 92.8 ± (2.797)*(0.52) = (90.28, 95.32).

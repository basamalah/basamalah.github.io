21. twice a number is twenty eight?​


Answer:

2N=28

Step-by-step explanation:

2N=28 good answer

To solve this problem, we can use algebra. Let's call the unknown number "x". Then, we can write the problem as an equation:

2x = 28

To isolate x on one side of the equation, we need to divide both sides by 2:

2x/2 = 28/2

x = 14

Therefore, the number is 14. We can check our answer by plugging it back into the original equation:

2(14) = 28

This is true, so our answer is correct.

2. mga halimbawa ng labor only contracting


Ang labor only contracting ay sinasabing bawal sa batas, ayon sa Labor Code (Article 106). Ang labor only contracting ay ang pag-hire ng mga empleyado na gagawa ng trabaho para sa isang main employer ng isang middleman na walang kakayanan o kapital na gawin ang trabaho. Ayon din sa mga direktiba ng Department of Finance, ang isang middleman ay dapat may kapital na P5 milyon at mga kagamitan. Isang halimbawa nito ay kung saan ang laborer ay pinapagawa ng bahay ng isang contractor, pero sya ay walang kapital at gamit. Ang gamit ay pwedeng galing sa laborer mismo o sa may ari ng bahay.  

Ang job contracting namana ng isa pang uri ng contracting. Ang term na ginagamit ngayon dito ay outsourcing. Ilan sa mga halimbawa nito ay ang mga call center.

9. What is the difference between a legitimate contracting and the prohibited labor only contracting?


Answer:

Job Contracting is valid and recognized by law, while Labor only contracting is a prohibited act. A finding that a contractor is a labor-only contractor is equivalent to a declaration that there is an employer-employee relationship between the principal and the employees of the labor-only contractor.

Iskemang Subcontracting

Ang iskemang subcontracting ay isang termino na ginagamit sa larangan ng negosyo na kung saan ang mga kumpanya ay kumukuha ng mga manggagawa o subcontractor para gampanan ang mga tungkulin o trabahong kinakailangan ng isang kumpanya. Halimbawa nito ay ang mga sumusunod:

Paghire ng mga kumpanya ng mga electricians Paghire ng mga kumpanya ng mga construction workers sa pagpapatayo ng mga gusali Paghire ng mga kumpanya ng mga plumbers para sa itinatayong kumpanya

Labor-only Contracting

Ang labor-only contracting ay isang termino na ginagamit naman upang ilarawan ang paguha ng mga subcontractor ng mga manggagawa para sa isang kumpanya.

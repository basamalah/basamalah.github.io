30. 2. In isosceles AACE, with base AC, MLA= 5x-1 and mzC=2x+19, find themeasure of each angle of the triangle. EAс​.


Since ∆ACE is an Isosceles triangle,

m∠A = m∠C

so,

5x - 1 = 2x + 19

solve for x,

5x - 1 = 2x + 19

5x - 2x = 19 + 1

3x = 20

x = 20/3

x = 6.67

Solve for ∠A = ∠C, so we can use any of them

∠C = 2x + 19

∠C = 2(6.67) + 19

∠C = 32.34°

Since ∠A = ∠C = 32.34°, let's find m∠E,

∠A + ∠C + ∠E = 180°

32.34° + 32.34° + ∠E = 180

∠E = 180° - 64.68°

∠E = 115.32°

So the angles are:

∠A = 32.34°

∠C = 32.34°

∠E = 115.32°

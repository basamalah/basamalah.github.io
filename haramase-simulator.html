9. what is in Continuous Simulation ?​


Answer:

In continuous simulation, the continuous time response of a physical system is modeled using ODEs, embedded in a conceptual model. The time response of a physical system depends on its initial state. The problem of solving the ODEs for a given initial state is called the initial value problem.

26. What is relative simulation?​


Answer:

Relative simulation is a type of simulation model that uses a relative scale to describe the behavior of a system. It is most commonly used in simulations of physical systems, such as those used in engineering, physics, and economics. In relative simulation, the objects in the system are not represented on an absolute scale, but rather on a relative scale. This means that the ratio of the size of one object to another can be changed to simulate different conditions. This type of simulation is useful for studying complex systems that have a wide range of behaviors and conditions.

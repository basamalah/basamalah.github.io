12. a rectangle has perimeter 4x + 8y + 6. saba says the sides can be 3 + and 3y + 2x. show that saba is right


A rectangle has four sides, so the total perimeter can be expressed as the sum of the lengths of the sides: 4x + 8y + 6 = (3 + 2x) + (3y + 2x) + (3 + 2x) + (3y + 2x)

Expanding the right-hand side and combining like terms, we get: 4x + 8y + 6 = 2x + 2x + 3 + 3 + 3y + 2x + 3y + 2x

This simplifies to: 4x + 8y + 6 = 10 + 6y + 4x

Both sides of the equation have the same value, so Saba is correct and the sides of the rectangle can be 3 + 2x and 3y + 2x.

21. a rectangle has perimeter 4x + 8y + 6. saba says the sides can be 3 + and 3y + 2x. show that saba is right


Answer:

Step-by-step explanation:

To show that Saba is right, we need to use the formula for the perimeter of a rectangle, which is given by:

Perimeter = 2 * (length + width)

In this case, the perimeter of the rectangle is 4x + 8y + 6, and Saba says that the length of the rectangle is 3 + and the width is 3y + 2x. Substituting these values into the formula for the perimeter, we get:

4x + 8y + 6 = 2 * (3 + + 3y + 2x)

Expanding the right side of the equation and simplifying, we get:

4x + 8y + 6 = 6 + 2 + 6y + 4x

Comparing the two sides of the equation, we can see that they are equal, which means that Saba's values for the length and width of the rectangle are correct. Therefore, Saba is right and the sides of the rectangle can be 3 + and 3y + 2x.

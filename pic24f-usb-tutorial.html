1. how to install windows 7 from a usb drive tutorial​


Answer:

What You'll Need

• A Windows 7 ISO or DVD

• Access to a computer with Windows 7, 8, 10, Vista, or XP installed and working properly, as well as with a DVD drive if you have a Windows 7 DVD

• A 4 GB (or larger) flash drive

1.) Create an ISO file from the Windows 7 DVD.

2.) Download Microsoft's Windows 7 USB/DVD Download Tool.

3.) Start the Windows 7 USB DVD Download Tool program, which is probably located in your Start menu or on your Start screen, as well as on your Desktop.

4.) On the Step 1 of 4: Choose ISO file screen, click Browse.

5.) Locate and select your Windows 7 ISO file, and then click Open.

6.) Click Next once you're back on the Step 1 of 4 screen.

7.) Click USB device on the Step 2 of 4: Choose media type screen.

8.) On the Step 3 of 4: Insert USB device screen, choose the flash drive or external hard drive you want to put the Windows 7 installation files on.

9.) Click the Begin copying button.

10.) Click Erase USB Device if you're prompted to do so on a Not Enough Free Space window. Then click Yes to the confirmation in the next window.

11.) On Step 4 of 4: Creating bootable USB device, wait for the program to format the USB drive and then copy the Windows 7 installation files to it from the ISO image you provided.

12.) The next screen you see should say Bootable USB device created successfully, with a Status of Backup completed.

13.) Boot from the USB device to start the Windows 7 setup process.

14.) You should have now installed Windows 7 by USB.

25. Question 3 how long will it take to transfer a file size of 1gb on a usb 2.0 and a usb 3.0 drive


GIVEN:

USB 2.0 and USB 3.0

File size = 1 GB (Gigabytes)

To find:

We need to find the time taken to transfer the file on a USB 2.0 and USB 3.0

Solution:

Given that, file size = 1 GB.

We know that 1 GB (Gigabytes) = 1024 MB (Megabytes).

1. In USB 2.0:

The speed of USB 2.0 is 60 MB/s (Megabytes/second).

To find the speed for 1024 MB, divide 1024 with 60.

\Rightarrow \frac{1024}{60}⇒

60

1024

\Rightarrow 17.0666⇒17.0666

\Rightarrow \approx 17⇒≈17

So, to transfer 1 GB of data, USB 2.0 will take 17 seconds approximately.

2. In USB 3.0:

The speed of USB 2.0 is 625 MB/s (Megabytes/second).

To find the speed for 1024 MB, divide 1024 with 625.

\Rightarrow \frac{1024}{625}⇒

625

1024

\Rightarrow 1.6384⇒1.6384

\Rightarrow \approx 2⇒≈2

So, to transfer 1 GB of data, USB 3.0 will take 2 seconds approximately.

Hence, the speed of USB 3.0 is greater than USB 2.0.

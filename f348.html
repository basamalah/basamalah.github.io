Here we will show you how to count by 7348, discuss counting by 7348 patterns, and tell you why knowing how to count by 7348 matters. To start off, note that Count by 7348 means counting in 7348s, or count by seven thousand three hundred forty-eights, and it is also called skip counting by 7348.

How to count by 7348

Normally, we would count by 1 like this: 1, 2, 3, 4, etc., but when we count by 7348, we count 7348, 14696, 22044, 29392, and so on.

In other words, to count in intervals of 7348 or skip counting by 7348, we start with 7348 and then add 7348 to get the next number, and then continue adding 7348 to the previous number to keep counting by 7348, like this:

7348

14696 + 7348 = 22044

22044 + 7348 = 29392

29392 + 7348 = 36740

You can of course skip count by 7348 forever, so it is impossible to make a list of all numbers, but below is a Count by 7348 Chart of the first 100 numbers to get you started.Count by 7348 Patterns

We organized the Skip Counting by 7348s Chart above in 10 rows and 10 columns so you can easily identify patterns.

Skip counting always creates patterns. Figuring out these patterns may help you if want to count by 7348, but don't have the Counting by 7348s Chart above. Obviously, one pattern with counting by 7348s is that the number increases by 7348.

Furthermore, if you look at each row above, each number in the row has the same last digit (ones place). That means that every tenth number has the same last digit.

If you look down the columns, you will see that the last digit (ones place) repeats itself in blocks of 5 over and over. The pattern of the last digit when you count by 7348 goes 8, 6, 4, 2, 0 and 8, 6, 4, 2, 0 and so on for as long as you count by 7348.

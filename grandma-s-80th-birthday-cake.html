10. find the 80th percentile ​


[tex]\huge\color{pink}{\boxed{\tt{ANSWER}}}[/tex]

40

The ordinal rank for the 80th percentile = (80/100) X 50 = 40. The next rank is 41 with 81 data value, so 81 is the 80th percentile. The ordinal rank for the 90th percentile = (90/100) X 50 = 45. The next rank is 46 with 87 data value, so 87 is the 90th percentile.

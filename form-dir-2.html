10. 2. Given: A UBDA DIR, UB = 28, BD = is the value of x? 36, UD = 43 and DR = 2x + 3. What is the value of x?​


Answer:

In triangle UBDA, we have:

UB + BD = UD + DA (By the Triangle Inequality Theorem)

28 + 36 = 43 + DA

64 = 43 + DA

DA = 64 - 43 = 21

Now, in triangle UDR, we have:

UD + DR = UR (By the Triangle Inequality Theorem)

43 + 2x + 3 = 21 + 28 (Substituting the values we found earlier)

2x + 46 = 49

2x = 49 - 46

2x = 3

x = 3/2

Therefore, the value of x is 3/2 or 1.5.

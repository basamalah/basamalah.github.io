7. 2. Find the variance and standard deviation for rollingsingle die.X012345A6P(X)1/61/61/61/61/61/61/6X²X-P(X)X²-P(X)​


Step-by-step explanation:

The variance and standard deviation can be calculated using the following formula:

Variance = E(X^2) - [E(X)]^2

Where E(X) is the expected value of X and E(X^2) is the expected value of X^2.

For a single die, the expected value of X is (0 + 1 + 2 + 3 + 4 + 5 + 6) / 7 = 21 / 7 = 3

And the expected value of X^2 is (0^2 + 1^2 + 2^2 + 3^2 + 4^2 + 5^2 + 6^2) / 7 = 91 / 7 = 13

Therefore, the variance can be calculated as:

Variance = 13 - (3)^2 = 13 - 9 = 4

And the standard deviation can be calculated as the square root of the variance:

Standard deviation = sqrt(variance) = sqrt(4) = 2.

So the variance is 4 and the standard deviation is 2.

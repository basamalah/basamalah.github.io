1. Distinguish between Inheritance and polymorphism? ​


Answer:

Inheritance:

Inheritance is one in which a new class is created that inherits the properties of the already exist class. It supports the concept of code reusability and reduces the length of the code in object-oriented programming.

Polymorphism:

Polymorphism is that in which we can perform a task in multiple forms or ways. It is applied to the functions or methods. Polymorphism allows the object to decide which form of the function to implement at compile-time as well as run-time.

Explanation:

1.  Inheritance is one in which a new class is created (derived class) that inherits the features from the already existing class(Base class). Whereas polymorphism is that which can be defined in multiple forms.

2. It is basically applied to classes. Whereas it is basically applied to functions or methods.

3. Inheritance supports the concept of reusability and reduces code length in object-oriented programming. Polymorphism allows the object to decide which form of the function to implement at compile-time (overloading) as well as run-time (overriding).

4. Inheritance can be single, hybrid, multiple, hierarchical and multilevel inheritance. Whereas it can be compiled-time polymorphism (overload) as well as run-time polymorphism (overriding).

5. It is used in pattern designing. While it is also used in pattern designing.

13. Cnidaria use sexual means of reproduction.


Answer:

Cnidarians use sexual means of reproduction.

Cnidarian reproduction both occur sexually and asexually.

Explanation:

The Asexual process

Cnidarians are strange-looking animals that really look more like plants, most live their lives stuck on the stable surface in the ocean. Those who stay this way usually have generations that go through the polyp and medusa stage such as the Obelia. Just like a plant which has leaves, flowers, and fruits, a Cnidarian can have feeding polyps that have stinging tentacles that are able to catch small prey, and a reproductive polyp that from time to time produces tine medusa forms that are able to swim away at the right time.  

The Sexual process:

After the mature medusa swims away, they then begin their own process of reproduction. Individual medusae will release either eggs or sperm which will meet.  

Some strange facts about the medusae:

Individuals can be male or female. They both have gonads that can produce both sperm and eggs.  

Once the egg is fertilized, it goes through these steps:

A Zygote is formed. It then turns into a Planula or larval stage. The Planula swims around and picks a spot then stays there. It then sticks to that surface and changes slowly into its polyp form. It eventually grows more polyps and the whole reproduction cycle starts again.

Cnidarians only have two body layers, the ectoderm and endoderm, separated by a jelly-like layer called the mesoglea. Most Cnidarians have radial symmetry.

28. What is POLYMORPHISM in JAVA Programming Panguage ?


The polymorphism principle is basically have a parent class and creating subclasses (children) that have almost the same functionality or are similar to the parent class but are unique in their own way.

For example you have an animal. There are many types of animals but they have some similar characteristics for example, let's say that all animals need to eat and sleep and they all have names. So the parent class, Animal, would have those eat and sleep methods.

public class Animal{
public void sleep();
public void eat();
}

But there are many types of animals such as birds, fish, etc. They have unique characteristics. So the children subclasses for example could look like this.

public class Bird extends Animal{
public void fly();
}

and a fish subclass could look like this...

public class Fish extends Animal{
public void swim();
}

The subclasses inherits the methods of Animal and add their own unique twist to it. :)

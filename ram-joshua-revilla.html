Angela is 35 yrs old while Joshua is 27 yrs old

Problem Solving

Statement of the Problem

Angelo is eight years older than Joshua. If sum of their ages is 62, find their ages.

Answer

Le t x be Joshua's age ; x = Joshua's age

It is stated that Angelo is 8 years older than Joshua.

Thus, x + 8 = Angelo's age

The sum of their ages id 62, hence,

Joshua's age + Angelo's age = 62

Substituting Joshua and Angelo's ages,

Joshua's age + Angelo's age = 62

x + (x + 8) = 62 ; add the variables

2x + 8 = 62 ; subtract both sides by 8

2x + 8 - 8 = 62 - 8 ; solve

2x = 54 ; divide both sides by 2

2x/2 = 54/2 ; solve

x = 27

Therefore, Joshua's age is 28.

Finding Angelo's age,

Angelo's age = x + 8

Angelo's age = 28 + 8

Angelo's age = 36

Therefore, Angelo's age is 36.

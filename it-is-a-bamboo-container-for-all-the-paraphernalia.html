6. a bamboo tube bank contains 245 coins, all 25 cents and 50 cents. How many of each kind are there if the amount in the bank is ₱72.50​


Answer:

200 pcs of 25 cents

45 pcsof 50 cents

Step-by-step explanation:

x + y = 245

0.25x + 0.5y = 72.5

multiply the second equaton by -4 since we want to cancel out x and get the value of y

-x - 2y = -290

solve the 2 equatons simultaneously

x + y = 245

-x - 2y = -290

cancel the x, multiply by -1

-y = -45

y = 45

substitute y to any of the 2 equations

x + y = 245

x + 45 = 245

x = 200

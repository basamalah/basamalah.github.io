28. Create a program that display average grade qualifed for Honor. AG >= 90 "with honor"​


Answer:

# Set minimum average grade for Honor

MIN_AVG_GRADE = 85

# Ask the user for their grades

num_grades = int(input("How many grades would you like to enter? "))

grades = []

for i in range(num_grades):

grade = float(input("Enter grade {}: ".format(i+1)))

grades.append(grade)

# Calculate the average grade

avg_grade = sum(grades) / num_grades

# Display the average grade

print("Your average grade is {:.2f}".format(avg_grade))

# Check if the average grade qualifies for an Honor

if avg_grade >= MIN_AVG_GRADE:

print("Congratulations, you qualify for an Honor!")

else:

print("Sorry, you do not qualify for an Honor.")

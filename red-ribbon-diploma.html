12. Rita has five more yellow ribbons than red ribbons. If twice the yellow ribbons is at most four times the red ribbons, at least how many red ribbons dies she have?


Answer:

There are at least 5 red ribbons

Step-by-step explanation:

Let y = number of yellow ribbons

Let r = number of red ribbons

 

y = r + 5

2y ≤ 4r

 

Substitute first equation into the second

 

2(r+5)  ≤ 4r   divide both sides by 2

r + 5  ≤ 2r     subtract r from both sides

5 ≤ r

 

r ≥ 5

26. In a box, the ratio of the number of red ribbons to the number of blue ribbons is 18:20. If there are 3 boxes, what is the ratio of the total number of red ribbons to the total number of the blue ribbons in 3 boxes?


If the ratio of the number of red ribbons to the number of blue ribbons in one box is 18:20, it means that for every 18 red ribbons, there are 20 blue ribbons. Since there are 3 boxes, the total ratio of the number of red ribbons to the number of blue ribbons in all the boxes is 3 * (18:20), which is equal to 54:60. This means that for every 54 red ribbons, there are 60 blue ribbons.

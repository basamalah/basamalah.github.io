5. solve by extracting the square root4e^2 – 99 = 1​


Answer:

e = 5, e = -5

Step-by-step explanation:

Solving via extracting the square root means that isolating the variable with the constant. This means that one part of the equation should have only contain variables, while the other side must have the constant.

[tex]4e^{2}  - 99 = 1[/tex]

The constant (-99) is still on the left side on the equation. We add 99 to both sides of the equation to isolate the variables from the constant. We also divide both sides of the equation by the coefficient of the variable.

[tex]4e^{2} -99+99 = 1+99\\\\4e^{2} = 100\\\\ e^{2} = 25[/tex]

We get the square root of both sides.

[tex]e^{2} = 25\\\\\sqrt{e^{2}}= \sqrt{25}[/tex]

We need to remember that the square root of a positive number can be a negative number or a positive number. For example:

[tex]-2*-2 = 4\\2*2=4[/tex]

This means that the square root of 4 is -2 or +2.

Going back to the original equation...

[tex]\sqrt{e^{2}} = \sqrt{25}\\\\e = 5, e = -5[/tex]

That means e = 5, or e =-5.

Extracting the square root is only a preffered option when solving for quadratic equations if the constant is a perfect square. If the constant is not a perfect square, completing the square, or using the quadratic formula is advised.

17. find the first derivative of the given function y=x^2+x^4e^(-2lnx)


This is actually pretty easy. It's just that the exponential and logarithmic functions make it seem difficult.

So the function is:
[tex]f(x)=x^2+x^4e^{(-2lnx)}[/tex]

We could rewrite this as:
[tex]f(x)=x^2+} \frac{x^4}{e^{(2lnx)} }[/tex]

Knowing the properties of an exponential raised to a logarithmic function, we could simplify this as:
[tex]f(x)=x^2+} \frac{x^4}{x^2} }[/tex]

How? [tex]2lnx=(lnx)^2[/tex] And [tex]e^{lnx}=x[/tex] Knowing those two properties, we were able to get to this answer.

See? This will be easy from now on.
[tex]f(x)= x^{2} + x^{2} [/tex]

Simplifying:
[tex]f(x)=2 x^{2} [/tex]

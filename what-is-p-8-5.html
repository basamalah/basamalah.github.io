Answer:

The equation 3(n+8) = -6(n+6) is an algebraic equation, which can be simplified by distributing the 3 and -6 on the left- and right-hand side respectively, we get:

3n + 24 = -6n - 36

Now, we can combine like terms:

3n - 6n = -6n + 3n = -12

24 = -36

Now we can add 12 to both sides to get:

36 = -24

Step-by-step explanation:

This equation is not true for any value of n because it is not possible for 36 to equal -24. Therefore, there is no solution for this equation and it is said to be inconsistent.

3(5 + 8) = -6(5 + 6)

3(13) = -6(11)

"What is P(8, 5) ?

Let's find the number of possibilities for the ordered set by applying the following formula:

[tex]nPr = \dfrac{n!}{\left(n-r\right)!}[/tex]

If: n = 8 and r = 5

Solving:

[tex]nPr = \dfrac{n!}{\left(n-r\right)!}[/tex]

[tex]8P5 = \dfrac{8!}{\left(8-5\right)!}[/tex]

[tex]8P5 = \dfrac{8!}{3!}[/tex]

[tex]8P5 = \dfrac{8*7*6*5*4*\diagup\!\!\!\!3!}{\diagup\!\!\!\!3!}[/tex]

[tex]8P5 = 8*7*6*5*4[/tex]

[tex]\boxed{\boxed{8P5 = 6720}}\end{array}}\qquad\checkmark[/tex]

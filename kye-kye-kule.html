Using this information, determine the function B(t) that represents the number of books in Kye's collection as a function of time

d(B(t))/dt = k*B(t)

d(B(0))/dt = k*B(0)

Solving for k, we find that k = d(B(0))/dt = d(100)/dt = 0.

Plugging this value of k back into the differential equation, we find that:

d(B(t))/dt = 0*B(t)

This simplifies to:

d(B(t))/dt = 0

This differential equation tells us that the derivative of B(t) is equal to 0, which means that the number of books in Kye's collection is not changing with time. Therefore, the function B(t) representing the number of books in Kye's collection as a function of time is a constant function, which means that the number of books in her collection remains the same over time.

Therefore, the function B(t) representing the number of books in Kye's collection as a function of time is:

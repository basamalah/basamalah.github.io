10. how will you solve problems involving fraction?​


Answer:

aku to si natuy na malapit na mayabag sa module

Answer:

Start by learning the terminology and fundamentals, then pratice adding, subtracting, multiplying, and dividing fractions.

Step-by-step explanation:

Add fractions with the same denominator by combining the numerators. To add fractions, they must have the same denominator. If they do, simply add the numerators together.

Subtract fractions with the same denominator by subtracting the numerators. If you need to subtract fractions, they must have the same denominator, just like if you were adding them. All you have to do is subtract the smaller numerator from the larger numerator to solve the problem.

Find a common multiple to add or subtract fractions without the same denominator. If the fractions don't have the same denominator, you'll need to find a common multiple of both denominators and convert each fraction so they have the same denominator. To do this, multiply both the numerator and denominator by the number that will convert it to the common multiple. Then, add or subtract the numerators to find the answer.

Multiply fractions straight across. Fortunately, multiplying fractions is pretty easy. If the fractions aren't already in the lowest terms, reduce them. Then, all you need to do is multiply the numerator by the numerator and the denominator by the denominator

Divide fractions by flipping the second fraction upside down and multiplying straight across. To divide fractions, start by making the fraction you want to divide by a reciprocal. Do this by turning it upside down so the numerator becomes the denominator and the denominator becomes the numerator. Then, multiply both numerators and both denominators together.

30. 1. How will you solve problems involving fractions?​


Step-by-step explanation:

Addition

For fractions with the same denominator just add the numerator and the denominator stays the same.

Example:

2/4+1/4=3/4

But if the equation has different denominator we must find first the least common denominator (LCD). and then we can now add.

Example:

2/3-3/2=4/6-9/6=-5/6

The same goes for subtraction, you just have to subtract it.

For multiplication, multiply the numerator for both the multiplier and multiplicand and the same goes with the denominator. If the answer can be in in the lowest term then so be it

Example:

2/4•2/5=4/20=1/16

For division just reciprocate the divisor and follow the rules of multiplication. that just easy

Example

2/4÷5/2= 2/4•2/5=4/20=1/16

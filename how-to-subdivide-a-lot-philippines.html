18. Your parents bought a 5km² lot for farming, it will be subdivided to five equal farms What will be the length of each plot if the width is 40m? What is the perimeter of each plot?​


Your parents bought a 5km² lot for farming, it will be subdivided to five equal farms What will be the length of each plot if the width is 40m? What is the perimeter of each plot?

Each lot will be 1km long and the perimeter of each lot will be 4km.

Step by step explanation:

1. Convert the width of each lot from meters to kilometers.

40m = 0.04km

2. Convert the area of each lot from hectares to square kilometers.

5km² = 0.005km²

3. Divide the length of each lot by the width of each lot.

1km / 0.04km = 25

4. Multiply the answer from step 3 by the width of each lot.

25 * 0.04km = 1km

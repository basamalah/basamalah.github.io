15. $7000 at 4% compound quarterly for 2 years ​


Answer:

$7580.00

Step-by-step explanation:

This problem involves compound interest. Of course, in these types of problems, we will need to use a formula.

We will use this formula:

[tex]A = P(1 + \frac{r}{n} ) ^{nt} [/tex]

From the problem, we have the given values:

P = $7000, r = 4%, n = 4, and t = 2.

We need to substitute these values into the formula and evaluate. The following series of equations will display and explain the full solution to solve this problem.

[tex]A = 7000(1 + \frac{0.04}{4} )^{(4)(2)} [/tex]

[tex]A = 7000(1.01)^{8} [/tex]

[tex]A = 7000(1.08286)[/tex]

[tex]A = 7580.00[/tex]

Thus, the compounded amount is $7580.00.

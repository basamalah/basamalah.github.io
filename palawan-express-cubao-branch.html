1. tree diagram there are 6 roads leading from antipolo to cubao and 3 roads leading from cubao to batangas. How many possible routes from antipolo to batangas through cubao


Answer:

18 possible routes

Step-by-step explanation:

In this problem, there are various of ways to solve this. One would be this.

First we label the places as

A = Antipolo

B = Batangas

C= Cubao

When there are x ways to do something and y ways to do another thing dependent to the first event, then we simply multiply these two values. That is xy = (6)(3)=18. Therefore there are 18 possible routes.

ALTERNATIVE EXPLANATION:

From A to C, we have 6 choices.

From C to B, we have 3 choices.

Now we have to find the number of possible routes from A to B but of course passing through C.

x = {1,2,3,4,5,6 }= routes from A to C

y={a,b,c}= routes from C to B

We start from A, we can take route 1 out of 6, then when we reach C, we can take route a. That is one possible route, namely route 1a.

Now for everytime we use the route 1, there are 3 possible routes to pair up with route 1, that is, route a, b, or c. So we have routes 1a, 1b, 1c.

It means that for every route from A to C, there are 3 possible routes. There are six routes, which makes it 6 x 3 = 18.

Listing all the elements and getting the cartesian product:

x·y= {1,2,3,4,5,6} · {a,b,c}

x·y= {1a,1b,1c,2a,2b,2c,3a,3b,3c,4a,4b,4c,5a,5b,5c,6a,6b,6c} = 18 routes.

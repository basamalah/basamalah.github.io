1. A local bookstore charges Php200 per ream of long bond papers for orders of less than 50, Php185 per ream for orders of 50 reams or more but fewer than 75, and Php160 per ream for orders of 75 reams or more. Which function best represents the total cost C if you are to buy x reams of long bond paper?​


Answer:

Piecewise Function:

              { 200x   if x < 50

       C = { 185x    if 50 ≤ x < 75

              { 160x   if x ≥ 75

Step-by-step explanation:

WHICH FUNCTION BEST REPRESENTS?Piecewise functions are used when a function requires more than one formula to produce the desired result.This describes when an input value crosses specific "boundaries," causing a rule or relationship to change.

In our given scenario, we have three boundaries:

For orders of less than 50, the bookstore charges 200php each.For orders of 50 or more but less than 75, each ream costs 185php.For orders of 75 and above, each ream costs 160php.

Before creating the function, one should first recall certain symbols for inequalities.

Greater than ( > )Greater than or equal to ( ≥ )Less than ( < )Less than or equal to ( ≤ )

We can now translate each condition:

Let x be the number of reams of long bond paper.

1.) For orders of less than 50, the bookstore charges 200php.

Translation: If x < 50, then each x is multiplied by 200.

2). For orders of 50 but less than 75, each ream costs 185php.

Translation: If 50 < x < 75, then each x is multiplied by 185.

Read as “if x is greater than or equal to 50 but less than 75”

3.) For orders of 75 and above, each ream costs 160php.

Translation: If x > 75, then each x is multiplied by 160.

Finalize our function:

      { 200x   if x < 50

C = { 185x    if 50 ≤ x < 75

      { 160x   if x ≥ 75

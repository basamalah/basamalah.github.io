4. A shortcut key that is use to close the currently open document or a file quickly


A shortcut key that is used to close the currently open document or a file quickly is CTRL + W

Explanation

This increasingly sophisticated technology can make it easier for us to do documentation from writing by hand or using old-fashioned typewriters, to using technology such as typing documents that we need for work, school, and others. Software that is offered to users to make their work easier is also supported by shortcuts that make it easier. When we have finished typing a document and want to close the file, there are two ways we can do it:

If you prefer to use the mouse, you can close the currently open document or a file by using the access toolbar in the right corner of the Microsoft display with the cross icon. If you prefer to type or use the keyboard, click Ctrl + W or click Ctrl + F4 (If Ctrl + F4 doesn't work, you may have to press the F-Lock or Fn key, then Ctrl + F4).

5. A shortcut key that allows the users to redo the last action performed in a file


A shortcut key that allows the users to redo the last action performed in a file is CTRL + Y

Explanation

Mistakes in typing or operating a software on a computer can happen anytime. For example, in copying a writing A and pasting it to another sheet, but we make a mistake by copying the wrong writing B. If this happens, don't panic because the software we use is sophisticated technology and makes it easier for users to use the software.

There are two ways that can be done:

If you want to repeat simple things like pasting and prefer to use the mouse, you can redo the action performed in a file by using the access toolbar in the left corner of the Microsoft display by clicking Undo Typing. If you prefer to type or use the keyboard, click Ctrl + Y or click F4 (If F4 doesn't work, you may have to press the F-Lock or Fn key, then F4).

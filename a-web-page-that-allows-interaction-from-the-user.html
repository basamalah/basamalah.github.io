1. A web page that allows interaction from the user


The answer is "Dynamic website".

There are only two types of web pages/site: The Dynamic website and Static website.  

Dynamic websites are the type of websites that contents are changing, depending on the user's actions, inputs (like keywords on their searches) and other means of interactions that they did upon using the site. For example, one user searched on Google using the keywords: "song lyrics". The list of websites will return to her by Google site with links related to song lyrics. Another example is a gaming website, wherein, users has different characters, and quests depending on how they play the game online.

Meanwhile static websites are with contents that are unchangeable or the contents are being displayed as is. Meaning no matter who or what the users do the website, all users may experience and see the same things.

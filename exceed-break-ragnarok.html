[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \large\underline{\mathbb{ANSWER}:} [/tex]

[tex] \qquad \LARGE \:\: \rm 23, \, 31, and \; 65 [/tex]

[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \large\underline{\mathbb{SOLUTION}:} [/tex]

Let x be the first number then write for the other two numbers.

[tex] \rm 1st \: number = x [/tex][tex] \rm 2nd \: number = x + 8 [/tex][tex] \rm 3rd \: number = 2(x+8)+3 [/tex]

Find x when their sum is 119.

[tex] x + (x+8) + \big[2(x+8)+3\big] = 119 [/tex]

[tex] x + x+8 + 2x+16+3 = 119 [/tex]

[tex] x + x+ 2x+8+16+3 = 119 [/tex]

[tex] 4x+27 = 119 [/tex]

[tex] 4x = 119 - 27 [/tex]

[tex] 4x = 92 [/tex]

[tex] \frac{4x}4 = \frac{92}4 \\ [/tex]

[tex] x = 23 [/tex]

Therefore, the first number is 23. Solve for the other two numbers.

Second Number:

[tex] \rm 2nd \: Number = x + 8 [/tex]

[tex] \rm 2nd \: Number = 23 + 8 [/tex]

[tex] \rm 2nd \: Number = 31 [/tex]

Third Number:

[tex] \rm 3rd \: Number = 2(x + 8) + 3 [/tex]

[tex] \rm 3rd \: Number = 2(31) + 3 [/tex]

[tex] \rm 3rd \: Number = 62 + 3 [/tex]

[tex] \rm 3rd \: Number = 65 [/tex]

Therefore, the other two numbers are 31 and 65

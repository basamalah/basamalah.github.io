30. Which of the following is an example/s of a linear communication model? a. Berlo's SMCR b. Both Shannon-Beaver and Berlo's SMCR c. Shannon-Weaver d. Both Shannon-Weaver and Berlo's SMCR e. Shannon-Beaver f. Both Shanon-Weaver and Berlo's SMCR g. Shanon-Weaver


The answer is D: Both Shannon-Weaver's Communication Model and Berlo's SMCR Model of Communication are examples of a linear communication model. It appears that the other choices just have misspellings but the answer is definitely the letter D. A linear communication model displays communication as going only one-way: from the sender to the receiver; hence, the term "linear."

Shannon-Weaver's Communication Model

The components of Shannon-Weaver's Communication Model are as follows:

SenderEncoderChannelDecoderReceiver

As the model displays, the action flow only starts from the sender and ends to the receiver. It does not show how the message goes back and forth with the receiver and sender roles being interchangeable. This type of model is linear.

Berlo's SMCR Model of Communication

As the acronym "SMCR" suggests,

S - SenderM - MessageC - ChannelR - Receiver

With this acronym, it is obvious that Berlo developed a linear communication model. Although it would have factors under each term, the arrows that signify the communication only goes from the sender to the receiver and not vice versa.

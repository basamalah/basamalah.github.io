2. A map is drawn to as scale of 3cm:6km.A bridge is 36km long.Calculate the length of the bridge on the map in centimetersTHINKPLANCARRY OUT THE PLANCHECK​


RATIO AND PROPORTION: MAP SCALE APPLICATION

Problem:

A map is drawn to as scale of 3 cm:6 km. A bridge is 36km long. Calculate the length of the bridge on the map in centimeters.

Answer with Step-by-step explanation:

In order for us to calculate the required value in the given word problem, let us identify the given values which we could use in solving the right answer.

The scale is read as  3 centimeters is to 6 meters which means that for  every 3 centimeters in the map is equivalent to 6 kilometers in the actual field.

By ratio and proportion, we can express this as:

[tex]\frac{3}{6} = \frac{x}{36}[/tex]

where: 36 km refers to the actual length of the bridge and x refers to the length of the bridge in the map.

Using algebraic manipulation:

3/6 can be further simplified to 1/2Multiply both sides by 36 to isolate the variable x in the other side:

[tex]\frac{36}{2}=x\\x = 18[/tex]

Therefore, the actual length of the bridge is 18 kilometers long.

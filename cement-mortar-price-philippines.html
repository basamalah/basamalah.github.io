11. 11. A 65-kg mortar shell has an impulse of 2.5 x 103 N s. Find the force of the mortar on the shell if it takes 0.2 s to leave the barrel of the mortar.​


Answer:

The impulse-momentum theorem states that the impulse on an object is equal to the change in momentum of the object. In this case, the impulse of the mortar shell is given as 2.5 x 10^3 N s, and the initial momentum of the shell is zero (since it was at rest before being fired).

Therefore, the final momentum of the shell is:

p = mv

where m is the mass of the shell and v is its final velocity. We can solve for v by using the formula for average acceleration:

a = (v - u) / t

where u is the initial velocity (which is zero in this case), and t is the time taken for the shell to leave the barrel (0.2 s).

Rearranging the equation, we get:

v = at

Substituting the given values, we get:

v = (2.5 x 10^3 N s) / (65 kg x 0.2 s) = 192.3 m/s

So the final momentum of the shell is:

p = (65 kg) x (192.3 m/s) = 12,500 kg m/s

The change in momentum of the shell is therefore:

Δp = p - 0 = 12,500 kg m/s

Since the time taken for the shell to leave the barrel is very short, we can assume that the force acting on the shell is constant during this time. Therefore, we can use the formula:

F = Δp / t

Substituting the values we have calculated, we get:

F = (12,500 kg m/s) / (0.2 s) = 62,500 N

So the force of the mortar on the shell is 62,500 N.

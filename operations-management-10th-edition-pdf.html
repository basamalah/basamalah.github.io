17. List 3 operations used to edit encoded texts


Answer:

Justify

Bold

Italic

Explanation:

Any text application has the above-mentioned commands that may be used in editing encoded texts.

1. Justify

We use this command to align the texts in the paragraph for both sides.

This command is widely used in editing encoded research papers/theses or dissertations. Texts in the paragraph become neat when this operation is applied. In the MS Word application, the shortcut key for justify is Ctrl + J.

2. Bold

This command is used when we want to stress some words in a given sentence or paragraph. The Bold command may also be used when encoding titles or subject headings.  In the MS Word application, the shortcut key for Bold is Ctrl + B.

3. Italic

When foreign texts are used, they have to be italicized. This indicates that the texts are from a different language or medium as what's used in the sentence or paragraph.  In the MS Word application, the shortcut key for Italic is Ctrl + I

18. PWEDE PA HELP!If ∆MSI ≅ ∆DEL, then the m∠S = 3x – 8 and m∠E = x + 16, find the value of x, m∠S and m∠E ?​


Since ∆MSI ≅ ∆DEL, we can conclude that their corresponding angles are congruent.

Therefore,

m∠M = m∠D (corresponding angles)

m∠S = m∠E (corresponding angles)

We are given that:

m∠S = 3x – 8

m∠E = x + 16

Therefore, we can write:

3x – 8 = x + 16 (since m∠S = m∠E)

2x = 24

x = 12

Substituting x = 12 in the expressions for m∠S and m∠E, we get:

m∠S = 3x – 8 = 3(12) – 8 = 28

m∠E = x + 16 = 12 + 16 = 28

Therefore, the value of x is 12,

m∠S is 28 degrees and m∠E is 28 degrees.

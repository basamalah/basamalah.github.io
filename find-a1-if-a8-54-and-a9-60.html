17. find a1 if a8=54 and a9=60?


Solving the problem of arithmetic sequences.

The value of a1 is 12.

What are arithmetic sequences?

A row of numbers with a specific pattern in the form of a summation that has the same or fixed difference or difference is known as an arithmetic sequence. A number with a fixed pattern based on addition and subtraction operations is called an arithmetic sequence. Sequences sometimes referred to as a counting row, has a certain pattern where the difference between any two subsequent tribes is constant and equal. In other words, each tribe in the arithmetic series (aside from the initial tribe) is obtained by adding a set number to the tribe before it.

This arithmetic sequence can be expressed by :

[tex]a, a+d, a+2d, a+3d, a+4d, ... , a+(n-1)d[/tex]

where:

a = initial number

d = common difference

Arithmetic Array Tribe

Suppose an is the nth row tribe, The formula for finding nth term of an arithmetic sequence

[tex]a_n = a + (n-1)d[/tex]

where

a = first term

n = number of tribes

d = common difference

Common Difference

the common difference, in the arithmetic array tribe, is the difference between two tribes. For example, d is the difference between tribes, mathematically it can be written as follows.

[tex]d = a_n-a_{n-1} = a_{n+1} - a_n[/tex]

The case: Find a1 if [tex]a_8 = 54[/tex] and [tex]a_9 = 60[/tex]

Given :

[tex]a_9 = 60\\a_8 = 54[/tex]

Find : [tex]a_1[/tex]

Solution:

Before finding the value of first term or a1, first determine the value of the common difference (d)

[tex]d = a_n-a_{n-1} = a_{n+1} - a_n[/tex]

[tex]d= a_9 - a_8\\d = 60 - 54 = 6.[/tex]

After the discovery of the value of the common difference we can find the value of a1 in two ways:

With direct calculations
Values from a9 = 60, a8 = 54, d = 6. then the value of the previous tribe is the value of the last tribe deducted by the common difference.

        [tex]a_9 = 60\\a_8 = 60 - 6 =54\\a_7 = 54 - 6 = 48\\a_6 = 48 - 6 = 42\\a_5 = 42 - 6 = 36\\a_4 = 36 - 6 = 30\\a_3 = 30 - 6 = 24\\a_2 = 24 - 6 = 18\\a_1 = 18 - 6 = 12[/tex]

By using formulas

        [tex]a_n = a + (n-1)d[/tex]

        [tex]a_n = a_9 = 60[/tex]

        n = 9
        d = 6

        [tex]a_9[/tex] = a + (9-1)6

        60 = a + 8*6
        60 = a +48
        a = 60 - 48
        a = 12

Thus, the a1 value of that arithmetic sequence is 12.

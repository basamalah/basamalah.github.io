28. Ano ang hulapi ng Langoy


Answer:

Ito ay salitang ugat.

Kung hahanapin ang hulapi, pwede ang salitang Langoyin

Answer:

Ang mga Halimbawa ng Unlapi, Gitlapi, Hulapi at kabilaan gamit ang mga salitang ugat na sayaw at gunita

ay ang mga sumusunod:

Explanation:

unlapi:

I + sayaw = isayaw

I + gunita = igunita

gitlapi:

S + in + ayaw = sinayaw

G + in +unita = ginunita

hulapi:

Sayaw + an = sayawan

Gunita + hin = gunitahin

kabilaan:

Sa + sayaw + in = sasayawin

I +g +in + gunita = iginunita

Ang panlapi ay isang morpema na ikinakabit sa salitang-ugat para makabuo ng isa pang salita.

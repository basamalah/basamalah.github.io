Answer:

To determine the x-intercepts and y-intercepts of a polynomial equation, we need to find the values of x and y where the equation equals zero.

Given the equation y = (x+4)(x+2)(x-1)(x-2)

To find the x-intercepts, we need to set y to zero and solve for x: y = (x+4)(x+2)(x-1)(x-2) = 0

x+4 = 0, x+2 = 0, x-1 = 0 and x-2 = 0

x = -4, x = -2, x = 1 and x = 2

The x-intercepts are (-4,0), (-2,0), (1,0), and (2,0)

For the y-intercept, we need to set x to zero and find the value of y: y = (x+4)(x+2)(x-1)

(x-2) = (0+4)(0+2)(0-1)(0-2) = 8

The y-intercept is (0,8)

In the case of the other equation y = (x+3)(x+2)(x-1)(x-2) x-intercepts are (-3,0), (-2,0), (1,0), (2,0) and y-intercept is (0,12)

Note that the x-intercepts and y-intercepts are the points at which the graph of the function crosses the x-axis or y-axis respectively.

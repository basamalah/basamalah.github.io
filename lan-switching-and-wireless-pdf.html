8. Difference between network diameter and collision domain relative to 10BASE-T. Why is the concept of collision domain not applicable to switched Ethernet to LAN?


Answer:

10Base-T uses a larger collision domain diameter than 100Base-T and thus has a different signaling system. The network diameter is the cabling distance between the two farthest points in the network. The diameter between two devices would be the cabling distance between the two devices.

23. Differentiate/compare the following components below. (5 points each) 1. DDR and SDR 2. HDR and SSD 3. USB flash drive and External Hard Drive 4. LAN and wireless connection 5. AMD and Intel​


Answer:

1. DDR and SDR: DDR (Double Data Rate) is a type of memory technology used in computer systems for high-speed data transfer. It operates at a higher frequency compared to SDR (Single Data Rate) and allows for faster processing times. SDR, on the other hand, is an older technology that operates at a slower speed.

2. HDR and SSD: HDR (High Dynamic Range) refers to a display technology that can display a wider range of colors and brightness compared to traditional displays. It provides a more vivid and lifelike image. An SSD (Solid State Drive) is a type of data storage device that uses flash memory to store data. It operates faster and more efficiently compared to traditional hard drives.

3. USB flash drive and External Hard Drive: A USB flash drive is a small, portable data storage device that uses flash memory and connects to a computer via USB. An external hard drive is a larger, standalone data storage device that also uses spinning disks to store data and connects to a computer via USB or other interface.

4. LAN and wireless connection: A LAN (Local Area Network) is a computer network that connects devices within a limited geographic area, such as a home or office. A wireless connection, on the other hand, allows devices to connect to a network without the use of physical cables.

5. AMD and Intel: AMD (Advanced Micro Devices) and Intel are two of the largest manufacturers of microprocessors for computer systems. AMD produces processors for both personal computers and servers, while Intel mainly focuses on the personal computer market. Both companies offer different processor models with varying levels of performance and power consumption.

30. 4 steps in testing local area network lan card and wireless network 4 steps in checking the network configuration ​


Answer:

1. Testing the Local Area Network (LAN) card:

Step 1: Check the physical connections to ensure that the LAN card is securely connected to the computer and to the network cable.

Step 2: Use the "ipconfig" command in the Command Prompt to check the IP address and other network settings of the LAN card.

Step 3: Run a ping test to check the connectivity to other devices on the network.

Step 4: Use the "tracert" command to check the route taken by the network packets and identify any network bottlenecks.

2. Checking the Wireless Network:

Step 1: Check the physical connections to ensure that the wireless adapter is securely connected to the computer.

Step 2: Use the "ipconfig" command in the Command Prompt to check the IP address and other network settings of the wireless adapter.

Step 3: Scan for available wireless networks and connect to the desired network.

Step 4: Use the "ping" command to check the connectivity to other devices on the network and use the "tracert" command to check the route taken by the network packets and identify any network bottlenecks.

Verify the device driver: Make sure that the LAN card has the correct device driver installed and that it is up-to-date.

Testing a Wireless Network:Check the physical connections: Make sure that the wireless access point or router is properly configured and that the antenna is securely attached.

Verify the device driver: Make sure that the wireless card has the correct device driver installed and that it is up-to-date.

Run a diagnostic test: Use the built-in diagnostic software that came with the wireless card to test its functionality.

Check the wireless network configuration: Make sure that the wireless network is configured correctly and that the correct wireless network is selected and connected to.

Checking the Network Configuration:Verify the IP configuration: Make sure that all the devices on the network have been assigned valid IP addresses and that they are configured correctly for the network.

Check the network settings: Make sure that the network settings are configured correctly for the network and that the network is properly configured for the devices that will be connecting to it.

Verify network connectivity: Use the ping command to check for network connectivity between devices on the network.

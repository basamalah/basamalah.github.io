4. Which of the following is NOT an animation type?A Entrance B. Exit C. Emphasis D. Encore​


The answer is Encore.

Animation refers to the movement of an object in a slide show presentation. It helps the lectures or meetings to become more interesting, dynamic and lively. Microsoft Powerpoint is a one of the presentation application that could be used to create such file. Listed below are the known animation types in Powerpoint:

Entrance - how the object will appear in the slide.Exit - how the object will leave or disappear in the slide.Motion - the user will determine the path or direction of the object as it appears or disappear in the slide.Emphasis - the type of animation that would make the object known/distinguished in the slide.

Encore refers to a repetition of performance at the end of the show as requested by majorities.

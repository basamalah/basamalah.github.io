28. Alvin, Betty and Cheryl scored a total of 2014 points during a competition. Betty scored 271 points less than Alvin. Betty scored 3 times as many points as Cheryl. How many points did Betty score? with explanation po pls


Answer:Let's start by assigning variables to the unknown quantities in the problem. We can let "A" be the number of points that Alvin scored, "B" be the number of points that Betty scored, and "C" be the number of points that Cheryl scored. Then we can set up a system of equations to represent the information given in the problem:

A + B + C = 2014 (the total number of points scored is 2014)B = A - 271 (Betty scored 271 points less than Alvin)B = 3C (Betty scored 3 times as many points as Cheryl)

Now we can use the third equation to substitute for "B" in the first two equations:

A + 3C + C = 2014 (substitute B = 3C into the first equation)A - 271 = 3C (substitute B = 3C into the second equation)

We can simplify the first equation by combining like terms:

A + 4C = 2014

Next, we can solve the second equation for "A" by adding 271 to both sides:

A = 3C + 271

Now we can substitute this expression for "A" into the first equation:

(3C + 271) + 4C = 2014

Simplifying, we get:

7C + 271 = 2014

Subtracting 271 from both sides:

7C = 1743

Dividing both sides by 7:

C = 249

So Cheryl scored 249 points. Now we can use the equation B = 3C to find out how many points Betty scored:

B = 3(249)

Simplifying, we get:

B = 747

Therefore, Betty scored 747 points.

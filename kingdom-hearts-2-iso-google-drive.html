6. how important is the ISO image creator and RUFUS software in creating portable bootable flash drive?​


Answer:

ISO files are ideal for distributing anything that would also be suitable for distribution on optical media. The main advantage of an ISO file is that it can be transferred over the network and the recipient can then 'burn' the ISO image to create their own CD.

Rufus is a utility that helps format and create bootable USB flash drives, such as USB keys/pendrives, memory sticks, etc. It can be especially useful for cases where: you need to create USB installation media from bootable ISOs (Windows, Linux, UEFI, etc.) you need to work on a system that doesn't have an OS installed.

29. how to collaborate using google drive?​


How to collaborate using google drive?​

Google Drive is one of the effective tools for collaboration tasks or work. It supports office documents (like word, excel and powerpoint) which could be work on by several people at a time and changes can reflect in real time. But before the collaboration begins, you must share the file to the authorized people only.  

To share a file or folder in Google Drive, just log in to your Google Drive account, click on the folder or file then click Share. You can input the email addresses of your authorized people to share your file.

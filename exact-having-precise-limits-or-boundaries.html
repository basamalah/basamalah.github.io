5. Difference between exact,precise and accurate. Give examples​


Answer:

Accuracy is how close a value is to its true value. An example is how close an arrow gets to the bull's-eye center. Precision is how repeatable a measurement is. An example is how close a second arrow is to the first one (regardless of whether either is near the mark).

28. Write, in words, the precise definition of a positive infinite limit.


Answer:

A positive infinite limit is defined as the limit of a function f(x) as x approaches infinity (i.e., x gets arbitrarily large) such that the values of f(x) become arbitrarily large and positive without bound, that is, for any positive number M, there exists a corresponding number N such that when x is greater than N, f(x) is greater than M. Symbolically, it can be expressed as lim f(x) = ∞ as x → ∞.

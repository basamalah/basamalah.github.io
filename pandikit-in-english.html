17. Formula puhunan + 15% ng puhunan = kabuoang presyo. Kabuoang presyo - puhunan = kinita. niyog 25.00, pandikit 12.00, barnis 21.00, palamuti 20.00​


Answer:

Ang puhunan ay:

25.00 (niyog) + 12.00 (pandikit) + 21.00 (barnis) + 20.00 (palamuti) = 78.00

Ang kabuuang presyo ay:

78.00 + (15% x 78.00) = 89.70

Ang kinita ay:

89.70 - 78.00 = 11.70

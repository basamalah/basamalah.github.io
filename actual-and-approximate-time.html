SOLVING FOR THE ORDINARY AND EXACT INTERESTS USING APPROXIMATE AND ACTUAL TIMES

Problem:

Find the interest on PHP  at 9.5% interest from Mar. 12, 2020 to Sept. 15, 2020, using the following:  

a,) ordinary interest using actual time.  

b.) ordinary interest using approximate time.

c.) exact interest using actual time.

d.) exact interest using approximate time​.

Answers with Step-by-step explanation:

In order to solve this problem about ordinary and exact interest, we need to know the difference between the two. In ordinary interest, we are using the 360 banker days. This means that for each month, only 30 days are counted within a year. On the other hand, the exact interest uses either 365 or 366 days (for leap year) which is the exact number of days in a year. So here, the year is 2020 which is a leap year, so in exact interest, we will be using 366 days.

For the approximate time, all months in a year are considered to have 30 days. For actual time, we really consider and sum up all the numbers of the days present within the given date range.

Now let us solve for the approximate time:

March = 30 - 12 = 18 daysApril to August = 30 days x 5 = 150September = 15 daysAdding all the days: 18 + 150 + 15 = 183 days

And for the actual time:

March = 31 - 12 = 19 daysApril and June = 30 days x 2 = 60 daysMay, July and August = 31 days x 3 = 93 daysSeptember = 15 daysAdding all the days: 19 + 60 + 153 + 15 = 187 days

Then, in order to solve for the interest, we just need the formula:

i = Prt

where: i refers to the interest, P refers to the principal amount, r refers to the interest rate and t refer to the time

@ Solving for ordinary interest using actual time:

[tex]ord_{actual} = (65,250)(0.095)(\frac{187}{360} )\\ord_{actual} = 3,219.91[/tex]

Therefore, the ordinary interest using actual time is PHP 3,219.91.

@ Solving for ordinary interest using approximate time:

[tex]ord_{approx} = (65,250)(0.095)(\frac{183}{360} )\\ord_{approx} = 3,151.03[/tex]

Therefore, the ordinary interest using approximate time is PHP 3,151.03.

@ Solving for exact interest using actual time:

[tex]exact_{actual} = (65,250)(0.095)(\frac{187}{366} )\\exact_{actual}= 3,167.12[/tex]

Therefore, the exact interest using actual time is PHP 3,167.12.

@ Solving for exact interest using approximate time:

[tex]exact_{approx} = (65,250)(0.095)(\frac{183}{366} )\\exact_{approx}= 3,099.38[/tex]

Therefore, the exact interest using approximate time is PHP 3,099.38.

The difference between actual and approximate time

An approximate number, time, or position is close to the correct number, time, or position, but is not exact.example :12 months ina year has 30 days (360 days in a year).

Actual time means the real time in which an event takes place, with the reporting on or the recording of the event simultaneous with its occurrence. Example:a year is taken as composed of 365 days

About approximate time

Approximate time means the operator's reasonable estimate of time spent on a particular subject matter. For example, a training schedule may indicate that a course will last for a certain period of time, such as his 1-2 hours. The plan may also state that training in a particular subject may last "about 3 hours". However, we recognize that when training is actually delivered, it may take more or less time than stated in the training plan. This flexibility allows adjustments based on changing mine conditions and operations, including the needs and experience of the people being trained.

19. Adding and Subtracting Polynomials Worksheet Name: ________________ 1. (-x² - 2) + (2x² + 1) = 2. (2x² + 3) - (3 - 4x²) = 3. (2x³ + 3x²) - (x³ + 8) =


ADDING AND SUBTRACTING POLYNOMIALS

[tex]__________________________[/tex]

1.) (-x² - 2) + (2x² + 1)

= -x² - 2 + 2x² + 1

= x² - 2 + 1

= x² - 1

2.) (2x² + 3) - (3 - 4x²)

= (2x² + 3) + (-3 + 4x²)

= 2x² + 3 - 3 + 4x²

= 6x² + 3 - 3

= 6x²

3.) (2x³ + 3x²) - (x³ + 8)

= (2x³ + 3x²) + (-x³ - 8)

= 2x³ + 3x² - x³ - 8

= x³ + 3x² - 8

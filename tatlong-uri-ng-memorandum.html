3. Tatlong uri ng memorandum​


Ang memorandum o kilala rin bilang memo ay isang nakasulat na mensahe na karaniwang ginagamit sa trabaho. Ito ay mga notis na ibinibigay tungkol sa mahahalagang isyu tungkol sa kompanya o sa manggagawa.

Tatlong uri ng memorandum:

Memorandum para sa kahilingan - Ang layunin nito ay makakuha ng positibong tugon sa isang kahilingan. Ang ganitong uri ng memo ay dapat gumamit ng mapanghikayat na wika dahil ang pinakalayunin natin ay mapagbigyan ang ating kahilingan.

Memorandum para sa kabatiran - Ito naman ay ginagamit upang kumpirmahin sa sulat ang isang bagay na napagkasunduan sa salita. Ang mga ganitong uri ng memo ay madalas makikitang ginagamit sa mga sitwasyong nangangailangan ng kasunduan sa pagitan ng dalawa o higit pang partido.

Memorandum para sa pagtugon - Ang mga uri naman ng ganitong memo ay karaniwang ipinapamahagi upang makahanap ng mainam na solusyon sa mga problema.  Ang layunin nito ay maaaring mangalap ng mga makabagong ideya mula sa empleyado at iba pang bumubuo sa organisasyon para malutas ang isyung kinakaharap.

Narito ang mga dapat tandaan upang makabuo ng magandang memo:

Gawing tiyak at maikli ang paksa ng memo. Huwag maglagay ng mga hindi klarong linya upang maintindihan agad ng mambabasa.Ilagay kung tungkol saan ang memo.Gumamit ng tamang tono. Maging sensitibo sa mga gagamiting salita at isaisip kung sino ang iyong target na mambabasa.Piliin ang tamang communication channel. Isaisip kung mas mainam ba na gumamit ka ng email o iaabot na lang ng personal sa mga empleyado ang memo.

10. Tatlong uri ng memorandum ayon kay Bargo.


Answer:

1. Paunang Memorandum - Isang uri ng memorandum na isinulat o natanggap ng tanggapan o ng kumpanya upang tiyaking ang mga layunin ng gawain at mga tungkulin ng mga taong sumasang-ayon sa memorandum ay maihahambing.

2. Pagmamay-ari Memorandum - Isang uri ng memorandum na naglalarawan ng mga kasunduan sa pagitan ng dalawang partido na may kaugnayan sa detalyeng pang-aari.

3. Pasilidad Memorandum - Isang uri ng memorandum na naglalarawan ng mga kondisyon at tungkulin ng partido sa isang kasunduan. Dito ang mga layunin, kapasidad ng gawain, prinsipyo ng pagkilos, at iba pang bagay ng mga partido sa kasunduan ay binanggit.

27. Ano ang iba't ibang layunin ng tatlong uri ng memorandum? (Pagtugon, Kahilingan, Kabatiran - Ayon kay Dr. Darwin Bargo)


Answer:

Ayon kay Dr. Darwin Bargo, ang tatlong uri ng memorandum ay may iba't ibang layunin. Ang mga ito ay ang mga sumusunod:

Memorandum ng Pagtugon (Response Memorandum) - Layunin nitong magbigay ng tugon o kasagutan sa isang katanungan o sulat mula sa ibang ahensya o departamento. Halimbawa, kung may nagtanong tungkol sa patakaran ng kumpanya, ang memorandum ng pagtugon ay magbibigay ng kumpletong kasagutan sa tanong na iyon.

Memorandum ng Kahilingan (Request Memorandum) - Ito ay ginagamit upang magsumite ng kahilingan o pakiusap sa ibang ahensya o departamento. Halimbawa, kung mayroong kakailanganin na kagamitan sa isang tanggapan, ang memorandum ng kahilingan ay magbibigay ng kahilingan sa tamang departamento para sa pagbibigay ng kagamitan.

Memorandum ng Kabatiran (Information Memorandum) - Layunin nitong magbigay ng kabatiran o impormasyon sa mga kawani ng isang organisasyon. Halimbawa, kung mayroong mga pagbabago sa patakaran ng kumpanya, ang memorandum ng kabatiran ay magbibigay ng mga detalye tungkol sa mga pagbabago na ito sa lahat ng kawani ng kumpanya.

Answer:

Ang layunin ng memorandum ng Pagtugon ay upang ipaabot ang sagot sa isang katanungan o pawastusan. Ang layunin ng memorandum ng Kahilingan ay upang humiling ng impormasyon o payo mula sa iba. Ang layunin ng memorandum ng Kabatiran ay upang magbigay ng kaalaman o impormasyon sa iba tungkol sa isang sitwasyon o paksa.

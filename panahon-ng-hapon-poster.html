25. gumuhit ng poster na nagpapakita kung paano hinarap ng mga pilipino ang kahirapan at kagutuman noong panahon ng mga haponyung matinong sagot po ​


Answer:

Kahirapan sa Panahon ng Hapon

Kung ako ay guguhit ng isang poster na nagpapakita ng kahirapang naranasan ng ating mga kababayan noong panahon ng mga Hapon, ang iguguhit kong poster ay ang isang lalaking nakayuko sa isang lamesa habang nasa harapan nya ang isang gabundok na Mickey Mouse money, at sa harapan nito ay ang isang latang sardinas.

Noong panahon ng mga Hapon, dumanas ng sobrang kahirapan ang karamihan sa pamilyang Pilipino dahil sa mabilisang pagtaas ng mga presyo ng mga produkto, sanhi ng mabilisang inflation. Ayon sa kwento ng mga matatandang naabutan pa ang panahon ng mga Hapon, kailangan mo daw ng isang bayong na Mickey Mouse money para lamang makabili ng isang lata ng sardinas.

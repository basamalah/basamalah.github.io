Answer:

To solve these problems, we first need to find the population size. Since the problem doesn't state how many students there are, we'll assume that there are 8 students in the population.

To compute the population mean, we add up all the scores and divide by the population size:

Population mean = (10 + 5 + 15 + 18 + 20 + 8 + 12 + 5) / 8 = 93 / 8 = 11.625

Therefore, the population mean is 11.625.

To compute the population variance, we need to find the difference between each score and the population mean, square each difference, add up all the squared differences, and then divide by the population size:

Population variance = [(10 - 11.625)^2 + (5 - 11.625)^2 + (15 - 11.625)^2 + (18 - 11.625)^2 + (20 - 11.625)^2 + (8 - 11.625)^2 + (12 - 11.625)^2 + (5 - 11.625)^2] / 8

Population variance = 196.375 / 8 = 24.546875

Therefore, the population variance is 24.546875.

To compute the population standard deviation, we take the square root of the population variance:

Population standard deviation = sqrt(24.546875) = 4.954

Therefore, the population standard deviation is 4.954.

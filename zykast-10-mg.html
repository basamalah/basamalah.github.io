14. how many mg of a drug are in 10 capsuls if 6 capsules contain 1500 mg? show the solution ​


Answer:

To solve this problem, we can set up the following equation:

(number of capsules) * (milligrams per capsule) = total milligrams

We are given that 6 capsules contain 1500 mg, so we can substitute those values into the equation:

(6 capsules) * (milligrams per capsule) = 1500 mg

We are trying to find the number of milligrams per capsule, so we can solve for that by dividing both sides of the equation by the number of capsules:

(6 capsules) * (milligrams per capsule) / (6 capsules) = 1500 mg / (6 capsules)

This simplifies to:

milligrams per capsule = 1500 mg / 6 capsules

We can then perform the division to find the value of milligrams per capsule:

milligrams per capsule = 250 mg/capsule

Now that we know the number of milligrams per capsule, we can use that value to find the total number of milligrams in 10 capsules. We can set up the following equation:

(number of capsules) * (milligrams per capsule) = total milligrams

Substituting in the values we have, we get:

(10 capsules) * (250 mg/capsule) = total milligrams

Performing the multiplication, we find that the total number of milligrams in 10 capsules is 2500 mg.

So, the solution to the problem is that there are 2500 mg of the drug in 10 capsules if 6 capsules contain 1500 mg.

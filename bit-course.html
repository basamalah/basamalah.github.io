20. Can 1 bit of light bounce off another bit of life? ​


Answer:

Yes, one bit of light can bounce off another bit of light, but not directly, and the effect is very rare. Light is made out of small quantum objects called photons. When you turn on a lamp, the light bulb begins creating and emitting trillions upon trillions of photons. Photons are in a class of quantum particles known as bosons. Bosons are special because many bosons can occupy the exact same quantum state at the same time. Light being made of bosons is what makes a laser beam possible. A laser beam is a collection of many photons all in the same quantum state. In contrast, particles that are not bosons cannot occupy the same state at the same time. This is one of the effects that keeps the atoms in an object from collapsing to a single point. The principle that dictates that non-bosons cannot be in the same state is called the Pauli Exclusion Principle. Non-bosons are also called fermions. The fact that bosons such as light can occupy the same state means that they don't get in each other's way

29. A bit is a 0 or a 1. A six-bit string is a sequence of length six consisting of Ds and 1s. How many six-bit strings contain


Since a bit can only be 0 or 1, and there are six bits in each string, the total number of possible six-bit strings is 2 to the power of 6:

2^6 = 64

Now we need to find how many of these six-bit strings contain:

Exactly four 1s:

We can choose the positions of the four 1s in the string in (6 choose 4) ways, and for each position we can either have a 1 or a 0 in that position. Therefore, the total number of six-bit strings with exactly four 1s is:

(6 choose 4) * 2^2 = 15 * 4 = 60

At least four 1s:

There are two possibilities: either the string has exactly four 1s, or it has exactly five or six 1s. We have already calculated the number of strings with exactly four 1s, so we just need to calculate the number of strings with five or six 1s.

For five 1s, we can choose the position of the five 1s in (6 choose 5) ways, and for each position we can either have a 1 or a 0 in that position. Therefore, the total number of six-bit strings with five 1s is:

(6 choose 5) * 2^1 = 6 * 2 = 12

For six 1s, there is only one possibility: all six bits are 1. Therefore, the total number of six-bit strings with six 1s is:

1

Adding these up, we get:

60 + 12 + 1 = 73

Therefore, there are 73 six-bit strings that contain at least four 1s.

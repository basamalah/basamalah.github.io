James' Dad is 5 times older than James and James is twice as old as his sister Sara. In two years, the sum of their ages will be 58. How old is James now? 

Let use x , 2x , 10x , where in

x = Sara's age 10x = Their dad's age

2x = James age

since we will find the sum in 2 years we will add ( +2 ) in the algebraic expression. So our equations will be ;

Sara's age: x + 2 James Dad age: 10x + 2

James age: 2x + 2

now , add all the equation. ( Sara's Age ) + ( James Age ) + (James Dad Age )

Equation: (x + 2) + (2x + 2) + (10x + 2) = 58

Solution:

 (x + 2) + (2x + 2) + (10x + 2) = 58

  x + 2 + 2x + 2 + 10x + 2 = 58

  13x + 6 = 58

  13x = 52

Devide both sides to 13 to find the value of x.

13x = 52

13 = 13

x = 52

So the value of x is 4 where x = 4. Now substitute the value of x to algebraic expression that is given.

Sara's age: x James dad's age : 10x

James age: 2x

Then , substitute the value of x which is 4.

 Sara's age: 4 = 4 James Dad age: 10(4) = 40

James age: 2(4) = 8  

Now , the question is how old is James now? 8 years old.

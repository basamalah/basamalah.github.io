1. the dimensions of a school parking lot is 2x + 7 by x + 3. write a quadratic expression that represents the area of the parking lot


the normal way

Here's how you solve, starting with your equation:

2x + 7 = x - 3

First, subtract "a" because it is the lowest variable. You want to isolate x. You should now have:

x + 7 = 3

Now, subtract 7 on both sides to continue isolating

Therefore:

x = -4

the quadratic way

(2x + 7)(x + 3)

2xx + 2x * 3 + 7x + 7 * 3

2x ^ 2 + 6x + 7x + 21

ANSWER:

2x² + 13x + 21ANSWER :

What is Asked?

The dimensions of a school parking lot is 2x + 7 by x + 3.

Write a quadratic expression that represents the area of the parking lot.

In algebra, a quadratic equation is any equation that can be rearranged in standard form as ax^{2}+bx+c=0 where x represents an unknown, and a, b, and c represent known numbers, where a ≠ 0. If a = 0, then the equation is linear, not quadratic, as there is no ax^2 term.

Formula

ax^2 + bx + c = 0

a, b, c = known numbers, where a ≠ 0

x = the unknown

The Given Sides are 2x + 7 and x + 3

In finding the Quadratic Equation, the Equation must be in the form of ax^2 + bx + c = 0.

We need to change the equation 2x + 7 = x + 3 to the form of

ax^2 + bx + c = 0

Solution :

Multiply both sides

(2x + 7)(x + 3)

2x^2 + 6x + 7x + 21 = 0

Simplify

2x^2 + 13x + 21 = 0

Hence, the Quadratic Equation of 2x+7= x+3 in the form of

ax^2 + bx + c = 0 is 2x^2 + 13x + 21 = 0

18. A school bus travels at an average speed of 60km/h. the bus has 12 stops to make every morning and each stop takes 2min. assuming the entire journey covers a total distance of 15km, at what time must the bus leave the school in order for the bus driver to return school by 7:25am. (the bus starts and ends its journey at the school).​


Answer:

The answer is 6:46 a.m.

Step-by-step explanation:

The time it takes for the bus to complete the journey is 15 km / 60 km/h = 0.25 hours = 15 minutes.

Since each stop takes 2 minutes, the time it takes to make all the stops is 12 stops * 2 minutes/stop = 24 minutes.

The total time the bus driver spends on the road is 15 minutes + 24 minutes = 39 minutes.

To find the time the bus must leave the school, we need to subtract 39 minutes from 7:25 a.m., which is equal to 6:46 a.m.

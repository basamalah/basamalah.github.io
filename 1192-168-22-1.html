Answer:

Formula of Circumference is given by:

C = 2πr

where,

C = Circumference

π = 3.14

r = radius = 22 cm

Solve for C,

C = 2πr

C = 2(3.14)(22)

C = 6.28(22)

C = 138.16 cm

The circumference of the circle is 138.16 cm.

So the correct answer is letter a. 138.16 cm

The function f is one to one if for any x1, x2 in the domain of f, then f(x1) is not equal to f(x2). That is, the same y-value is never paired with two different x-values.

Example 1: The relation pairing an SSS member to his/her SSS number.

Solution:

Each SSS member is assigned to a unique SSS number. Thus, the function is one-to-one function.

Example 2:

The relation pairing a real number to its square.

Solution:

Each real number has a unique perfect square. Thus, the relation is a function. However, two different real numbers such as 2 and -2 may have the same square. Thus, the function is one-to-one.

15. what is the principal difference between hub routing and switch routing?


Answer:

The big difference between hub and switch is in the method in which frames are being delivered. In a hub, a frame is passed along or "broadcast" to every one of its ports. It doesn't matter that the frame is only destined for one port.

16. There are two routes Sally can take to travel home from school. Route A is 800 m. Route B is 1 km. What is the percentage decrease in distance if Sally took route A instead of route B?


Answer:

To find the percentage decrease in distance if Sally took route A instead of route B, we first need to calculate the difference between the distances of the two routes:

1 km = 1000 m

So, the difference in distance between Route A and Route B is:

1000 m - 800 m = 200 m

Next, we need to find the percentage decrease from Route B to Route A, which can be calculated using the formula:

% decrease = (difference / original value) x 100%

In this case, the original value is the distance of Route B (1000 m), and the difference is 200 m. Substituting these values into the formula, we get:

% decrease = (200 / 1000) x 100%

% decrease = 0.2 x 100%

% decrease = 20%

Therefore, if Sally took Route A instead of Route B, she would be traveling 20% less distance.

Answer:

The only one-to-one function is the relations pairing a distance d (in kilometers) traveled along a given jeepney route to the jeepney fare for traveling that distance.​

Step-by-step explanation:

What is a one-to-one function?

It is a function wherein every value substituted for x has an equal value expressed as y.

If the value of x can be equal to several values of y, it is not a one-to-one function.

So, looking at the two choices, only the relation between distance and jeepney fare is one-to-one. This is because whenever the value of distance changes, the fare also changes proportionally.

Unlike the relation of pairing a person, a person may be paired with anyone and it will not affect the value of either person. One person may also be paired with more than one person.

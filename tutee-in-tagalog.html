14. Write a system of linear inequalities to describe the situation in the given problem. Miss Mendoza has two tutees, one paying Php 250.00 per section and the other paying Php 300.00 per session. She must earn atleast Php 6000.00 a month to pay expenses while attending college. Write an inequality that shows the various ways she can schedule her time to achieve her goal, ​


Answer:

Let x be the number of sections Miss Mendoza tutors the student who pays Php 250.00 per section, and let y be the number of sessions Miss Mendoza tutors the student who pays Php 300.00 per session. We can write the following system of linear inequalities to describe the situation:

250x + 300y >= 6000

x >= 0

y >= 0

The first inequality represents the condition that Miss Mendoza must earn at least Php 6000.00 per month. The second and third inequalities represent the fact that the number of sections and sessions Miss Mendoza tutors must be nonnegative, since she cannot tutor a negative number of times.

This system of inequalities represents the various ways Miss Mendoza can schedule her time to achieve her goal of earning at least Php 6000.00 per month by tutoring students.

13. 2. Joelita is stacking game cards. When she has 1 level, she uses 2 cards. Completing the two-level stacks, she had used 7 cards. Then the patter continues. If Joelita used 122 cards for the base of the stack, how many levels of stacked cards did he make?​


Answer:
Therefore, Joelita made approximately 3 levels of stacked cards.

Step-by-step explanation:

Let x be the number of levels of stacked cards Joelita made.

When she has 1 level, she uses 2 cards. Thus, when she has x levels, she will use 2x cards for the base of the stack. For the additional levels, she will use 2 more cards than the previous level.

The total number of cards used for the base and additional levels is given by the sum of the arithmetic series:

122 + (2x + 2) + (2x + 4) + ... + (2x + 2(x-1))

= 122 + 2x(1 + 2 + ... + x) + 2(1 + 2 + ... + x-1)

= 122 + 2x(x+1)/2 + 2(x-1)(x)/2

= 122 + x^2 + 3x - 2x

= x^2 + x + 122

Since Joelita used 122 cards for the base of the stack and a total of 7 + 2x cards for the stack, we have:

122 + 7 + 2x = x^2 + x + 122

2x + 7 = x^2 + x

x^2 - x - 7 = 0

Using the quadratic formula, we get:

x = (-(-1) ± sqrt((-1)^2 - 4(1)(-7))) / (2(1))

x = (1 ± sqrt(29)) / 2

Since x must be a positive integer, we take the positive root:

x = (1 + sqrt(29)) / 2 ≈ 3.19

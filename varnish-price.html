The solution is very easy, namely to check first whether the old paint layer has completely dried before applying the new coating.

Here's the arranged procedures for finishing a project that involves varnishing, using the letters A to E:

A. Clean the parts to be varnished

B. Stain or color the project before varnishing

C. Choose the type of varnish to be applied

D. Apply the varnish by using a brush or spray

SOLVING FIRST ORDER LINEAR ODE BY LAPLACE TRANSFORMS

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

[tex] \large \mathbb{INSTRUCTION:}[/tex]

Solve the following ordinary differential equation using Laplace Transforms.

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

[tex] \large \mathbb{GIVEN:}[/tex]

[tex] \sf \: \frac{dy}{dx} + 0.4y = {3e}^{ - x} \: and \: \sf \: y(0) = 5(1)[/tex]

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

[tex] \large \mathbb{FORMULA:}[/tex]

[tex] \sf \: L( \frac{dy}{dx} ) = 5Y(s) - y(0) [/tex][tex]\sf \: L( {e}^{ - bx)} = \frac{1}{s + b} [/tex]

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

[tex] \large \mathbb{SOLUTION:}[/tex]

Then taking the Laplace Transform of both sides we get;

[tex] \sf \: L( \frac{dy}{dx} + 0.4y) = L( {3e}^{ - x)} \\ \\ \sf \: [s Y (s)-y(0)] + 0.4Y(s)= \frac{3}{s + 1}[/tex]

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

Using the initial condition, y(0)=5 we get;

[tex] \sf \: [s Y (s)-5] + 0.4Y(s)= \frac{3}{s + 1} \\ \\ \sf \: (s + 0.4)Y(s) = \frac{3}{s + 1} + 5 \\ \\ \sf \: (s + 0.4)Y(s) = \frac{5s + 8}{s + 1} \\ \\ \sf \: Y(s) = \frac{5s + 8}{(s + 1)(s + 0.4)}[/tex]

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

Writing the expression for Y(s) in terms of partial fraction.

[tex] \sf \frac{5s + 8}{(s + 1)(s + 0.4)} = \frac{A}{s + 1} + \frac{B}{s + 0.4 } \\ \\ \sf \frac{5s + 8}{(s + 1)(s + 0.4)} = \frac{As + 0.4A + Bs + B}{(s + 1)(s + 0.4)} \\ \\ \sf 5s + 8 = As + 0.4A + Bs + B[/tex]

[tex] = = = = = = = = ======= = = = = = = = = = = = = = = = [/tex]

[tex] \large \mathbb{ANSWER:}[/tex]

[tex] \sf \: A + B = 5 \\ \sf \: 0.4A + B = 8[/tex]

26. inverse laplace of 1/(s^2+4s+13)​


Solution:

[tex]\frac{1}{s^{2} + 4s + 13}[/tex]

Step 1: Solve by completing the square.

[tex]s^{2}+4s+4-4+13\\\\(s+2)^{2}+9[/tex]

We get,

[tex]\frac{1}{(s+2)^{2}+9 }[/tex]

Note: In this given, I don't have any similar denominator in the Laplace Transform Table so, I use method in the Step 1.

Step 2: In the Laplace Transform Table, we have,

[tex]e^{at}sin(at)=\frac{b}{(s-a)^{2}+b^{2} }[/tex]

[tex]\frac{1}{(s+2)^{2}+3^{2} }[/tex]

Note: Notice that the a in the numerator is positive and the b in the denominator is not the same in the numerator. For the a, sign will be negative. For the b, I can obtain the [tex]\frac{3}{(s+2)^{2}+3^{2} }[/tex] by multiplying 1/3.

We get,

[tex]\frac{1}{3} e^{-2t}sin(3t)[/tex]

Answer:

[tex]\frac{1}{3} e^{-2t}sin(3t)[/tex]

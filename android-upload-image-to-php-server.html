16. Ito ang proseso ng pagkuha ng isang electronic file gaya ng text, image, music, o video file mula sa web server. A. Upload b. Download C. Click d. Double-click


Sagot

B - Download. Ang proseso ng pagkuha ng digital file/s sa isang web server.

Bakit hindi...

A - Upload. Ang pagpapadala ng digital file sa web server galing sa isang PC. Kasalungat nito ang Downloading.

C - Click. Tawag sa pagpindot ng mouse button. Ang click ay kadalasang ginagamit upang mag-select sa Computer.

D - Double-click. Mabilis Dalawang beses na pagpindot ng mouse button. Ang double-click ay madalas na ginagamit upang mag-execute ng commands tulad ng pagbukas ng files.

Ang download ay iba sa Data Transfer. Ang download ay pagkuha ng files mula sa isang server na naka-konekta sa internet habang ang Data Transfer ay palitan o paglipat ng mga files ng dalawang storage devices.

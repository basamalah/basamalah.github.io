[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \Large\underline{\mathbb{QUESTION}:} [/tex]

The following are the values of a, b, and c that Francine and Czarina got when they expressed 3 = -5x² + 3x in standard form:

Francine [tex] \sf a = \text-5, \, b=3, \, c= 3 [/tex]Czarina: [tex] \sf a = 5, \, b= \text-3, \, c=3 [/tex]

» Who do you think got the correct values of a, b, and c?

A. Czarina B. Francine C. Both Czarina and Francine D. Nobody got the correct values

[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \Large\underline{\mathbb{ANSWER}:} [/tex]

[tex] \qquad \LARGE \rm{A. \: Czarina} [/tex]

[tex]••••••••••••••••••••••••••••••••••••••••••••••••••[/tex]

[tex] \Large\underline{\mathbb{EXPLANATION}:} [/tex]

» We can rearrange the quadratic equation into its standard by transposing, by simplifying, and by the commutative property.

» Follow the standard form of ax² + bx + c = 0. Transpose the following terms and change their signs.

[tex] \rm 3 = \text-5x^2 + 3x [/tex]

[tex] \rm 5x^2 - 3x + 3 = 0 [/tex]

» Transposing the terms -5x² and 3x to the left side of the equation will give us 5x² and -3x. By getting the values of the constant and coefficients, we will get:

[tex] \rm a = 5 [/tex][tex] \rm b = \text-3 [/tex][tex] \rm c = 3 [/tex]

» In this equation, we can say that Czarina was correct. But what if Francine was also correct?

» The leading coefficient a can be any number as long as it is non-zero. So we can include negatives. Multiply both sides by -1.

[tex] \rm (5x^2 - 3x + 3)(\text-1) = 0(\text-1) [/tex]

[tex] \rm \text-5x^2 + 3x - 3 = 0 [/tex]

» By getting the values of a, b, and c, we will get.

[tex] \rm a = \text-5 [/tex][tex] \rm b = 3 [/tex][tex] \rm c = \text-3 [/tex]

» Close, but Francine got the wrong answer because her constant value is 3, not -3. Thus, we can say that Czarina is only correct on getting the values.

15. 1. Roller blades regularly sell for PhP 1 850.00 in Francine's store. Find the sale price if Francine offers a discount of 20%. full sollution.​


So, regular price of roller blade is 1850Php.

Francine offers 20% discount, it means, that price becomes 20% less.

The initial price is 100%, so if it becomes 20% less it is 100% – 20% = 80%.

So, the selling price is 80% from the initial price, let's calculate it:

[tex]1850 \times \frac{80\%}{100\%} = 1850 \times 0.08 = Php1480[/tex]

So, the sale price is Php 1,480

19. In a basketball game, Jane and Francine made 20 baskets. Jane scored 4 times as many baskets as francine. how many baskets did Jane score​


Answer:

Jane's score is 16.

Step-by-step explanation:

Let x be Francine's score and 4x be Jane's.

[tex]\\ \sf{} 4x + x = 20[/tex]

Find Francine's score first:

[tex]\\ \sf{} 4x + x = 20[/tex]

[tex]\\ \sf{} 5x = 20[/tex]

[tex]\\ \sf{} x = \frac{20}{5} [/tex]

[tex]\\ \large \boxed{ \sf{} x = 4} \\ \\ \\ [/tex]

Francine's score is 4.

Now that we've found Francine's score, let's find Jane's:

[tex]\\ \sf{} = 4x[/tex]

[tex]\\ \sf{} = 4(4)[/tex]

[tex]\\ \large \boxed{\sf{} = 16} \\ \\ [/tex]

Jane's score is 16.

Check to verify:

[tex]\\ \sf{} 16 + 4 = 20[/tex]

[tex]\\ \sf{} 20 = 20 \\ \\ \\ [/tex]

Number Logic:

The answer is 73.

Solution:

Find the value of the 3 witches with broom and wand on their hands using x.  

3x = 45

3x/3 = 45/3

x = 15

Therefore, each witch with broom and wand on her hand is equal to 15.

Find the value of the wand by using w.

3w = 21

3w/3 = 21/3

w = 7

Therefore, each wand is equivalent to 7.

Now, find the value of the broom using b. Note that there are 4 brooms in the picture which makes

4b = 12

4b/4 = 12/4

b = 3

Therefore, each broom is equivalent to 3.

Using all the values obtained, find the value of a broom + a witch with no broom and wand on her hand x a double wand which makes

3 + 15 (7 + 3) x (2x7)

3 + 15 - 10 x 14

3 + 5 x 14

By MDAS rule:

3 + 70 = 73

Therefore, broom + a witch with no broom and wand x a double wand is equals to 73.

Kris-3400 Lexi-P5200 Cynthia-P8900 C. Kris-3500 Lexi - 5300 Cynthia- 9000 D. Kris-3600 Lexi- 5400 Cynthia- 9000(Nonsense-ReportRight Answer- 60 pts.)​


Calculating Ratio of a Number

Answer:

1. B. 16

2. C. 15

3. D. Kris - 3600 Lexi- 5400 Cynthia - 9000

Step-by-step explanation:

1. Find the total number of parts in the ratio.

   Example: In a class of 40, the ratio of boys to girls is 3:2. How many are girls?

                   Ratio  3:2

                   Total number of parts = 3+2

                                                         = 5

2. Divide the total number by the total number of parts. This will tell us how much each part is.

    Example : In a class of 40, the ratio of boys to girls is 3:2. How many are girls?

                      One part = total number / total number of parts

                                      = 40 / 5

                                      = 8

3. Multiply the value of one part by each number in the ratio.

   Example:    Ratio: Boys: Girls

                                 3:2

                      ***One part = 8

                      Number of boys: 3 x 8 = 24

                      Number of girls: 2 x 8 = 16

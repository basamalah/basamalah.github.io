1. An office manager booked 55 airline tickets. He booked 7 more tickets on American airlines than United airlines. On Southwest airlines he booked 4 more than twice as many tickets as on United. How many tickets did he book on each airline?


Let x - ticket for United airlines
7 + x - ticket for American airlines
4 + 2x - ticket for Southwest

x + (7 + x) + (4 + 2x) = 55
4x + 11 = 55
4x = 55 - 11
4x = 44
x = 11 tickets for United airlines
7+x = 18 tickets for American airlines
4+2x = 26 tickets for Southwest airlines

To check, just add all the tickets and u will get 55 tickets.

2. An office manager booked 72 airline tickets. He booked 6 more tickets on Airlines A than Airlines B. On Airlines​ C, he booked 6 more than thrice as many tickets as on Airlines B. How many tickets did he book on each​ airline?


Tickets sold:

Airline A = 18

Airline B = 12

Airline C = 42

Step-by-step explanation:

Let:

A = tickets sold from Airline A

B = tickets sold from Airline B

C = tickets sold from Airline C; WHERE,

B = A - 6

C = (3B) + 6 = [3(A-6)] + 6 = (3A-18) + 6 = (3A - 12)

The purpoae of above solutions is to reduce the variables into just one (which is A).

Now, solve for A using the main equation:

A + B + C = 72

A + (A-6) + (3A-12) = 72

A + A + 3A - 6 - 12 = 72

5A - 18 = 72

5A = 72 + 18

5A = 90

5A/5 = 90/5

A = 18

Now that we knew the # of tickets sold from Airline A, we can now solve tickets sold for Airlines B & C:

B = (A - 6) = (18 - 6) = 12

C = 3B+6 = 3(12)+6 = 42

CHECKING:

The total number of tickets sold must sum up to 72:

A + B + C = 72

18 + 12 + 42 = 72

72 = 72

17. Joan is buying airline tickets for her family. Each airline ticket costs $232 with a baggage fee of $13. The airline charges a booking fee of $45 for the entire set of tickets. Joan spent a total of $1,515. If x represents the number of tickets, which equation can be used to find the value of x?


Answer:

245x+45=1515

Step-by-step explanation:

Given :Joan is buying airline tickets for her family. Each airline ticket costs $232 with a baggage fee of $13. The airline charges a booking fee of $45 for the entire set of tickets. Joan spent a total of $1,515.

To Find:If x represents the number of tickets, which equation can be used to find the value of x?

solution:

Since each ticket costs $232 with a baggage fee of $13.

So, cost of 1 ticket is 232+13=245

Cost of x tickets = 245x

The airline charges a booking fee of $45 for the entire set of tickets.

So, total cost will be 245x+45

We are also given that Joan spent a total of $1,515.

⇒245x+45=1515

Thus the above equation can be used to find the value of x

9. analyze the steps in doing the following activities save save as adobe pdf​


Answer:

To save a file in .pdf format:

1.Open the document you want to save as a .pdf

2.Click the File tab

3.Click Save As (choosing Save As keeps your original version and allows you to save an additional copy in another file format.)

4.In the File Name box, enter a name for the file if you haven't already

5.Click the drop down arrow in the Save as Type box and click PDF (*.pdf) - If you don't see this as an option in the menu, make sure you have the free Adobe Reader software installed on your computer.

6.If you want the file to open in the selected format after saving, select the Open file after publishing check box

7.Click Options box for other available options

8.Click Save

**You may also see an option to "Export" your document/file to a PDF in your File menu, which you an use instead of following the "save as" instructions if you prefer.

26. Why it is hard to destroy adobe flash?


Answer:

Adobe Flash was once a popular multimedia platform used for creating animations, videos, and interactive applications on the web. However, over the years, it has become outdated and insecure, with numerous security vulnerabilities and performance issues.

Despite its flaws, Adobe Flash has remained difficult to destroy for a few reasons. Firstly, it was widely adopted and used by many websites and applications, meaning that many developers invested time and resources into building Flash-based content. As a result, simply phasing out Flash overnight would have resulted in significant disruption and broken functionality across the web.

Secondly, Flash is deeply embedded into the architecture of many legacy systems, making it difficult to remove without causing compatibility issues. Many software applications and tools depend on Flash, and removing it could result in those applications no longer functioning properly.

Lastly, Adobe Flash was a proprietary technology, meaning that Adobe had full control over its development and distribution. While Adobe announced the end of support for Flash at the end of 2020 and recommended users to uninstall it, some users may still be using Flash due to its continued availability or necessity for certain applications or content.

Overall, the slow and gradual process of phasing out Flash has been necessary to ensure a smooth transition to more modern, secure, and efficient technologies.

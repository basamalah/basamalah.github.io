9. a coconut fell on the tree if the coconut tree falls for 2.5 seconds what will be the final velocity​


Given: Vi = 0 m/s

a = 9.81 m/s²

t = 2.5 s

Note: The coconut was at rest that's why the initial velocity is 0m/s and it's falling to the ground so the acceleration will be due to gravity with the constant value of 9.81m/s².

Unknown: Vf = ?

Formula: Vf = Vi + at

where in, Vf = final velocity

Vi = initial velocity

a = acceleration

t = time

Solution: Vf = Vi + at

= 0m/s + (9.81m/s²)(2.5s)

= 0m/s + 24.5m/s

= 24.5m/s

Final Answer: Vf = 24.5 m/s

13. the ratio of the height of the banana tree to the coconut tree is 2:3. The banana tree is 2.5 m tall. What is the height of the coconut tree?


Problem:The ratio of the height of the banana tree to the coconut tree is 2:3. The banana tree is 2.5 m tall. What is the height of the coconut tree?

Solution:

Let x be the height of the coconut tree

2:3 = 2.5:x

2/3 = 2.5/x

2x = 3(2.5)

2x = 7.5

x = 7.5/2

x = 3.75

Answer:

The height of the coconut tree = 3.75 m

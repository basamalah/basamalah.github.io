1. A villa has a rectangularswimming pool whose area is(x2 + 11x - 26)m? What are thedimensions of the swimmingpool? *​


(x + 13) m and (x - 2) m.

Step-by-step explanation:

Given the area of the rectangular pool, perform factorization in order to find the dimensions.

With the area (x² + 11x - 26)m², we can rewrite this as factors.

[tex] {x}^{2} + 11x - 26[/tex]

To find the factors, find two numbers whose sum equals b, which is 11x, and whose product equals c, which is -26.

These two numbers are 13 and -2. We can rewrite now the polynomial with factors (x + 13) and (x - 2).

[tex](x + 13)(x - 2)[/tex]

Since they are factors, they can be treated similarly to the formula for the area of a rectangle.

[tex]a = lw[/tex]

The variable a stands for the area, while l and w stand for the length and width respectively.

Meaning, if the area is (x² + 11x - 26) m², then the length is (x + 13) m and the width is (x - 2) m, or vice versa. If we let (x + 13) m to be the width, then the length to be (x - 2) m, it will still yield the same area of (x² + 11x - 26) m².

Preferably, let (x + 13) m be the length and (x - 2) m be the width, since (x + 13) m is longer than (x - 2) m.

Therefore, the dimensions of the rectangular pool are (x + 13) m and (x - 2) m.


2. the length of a swimming pool in villa mila is 8 m longer than its width and the area is 105 m2


Answer:

Width = 7 meters

Length = 15 meters

Step-by-step explanation:

Formula:

A = lw

Let x be the width

Let x + 8 be the length

105 = (x)(x+8)

105 = x² + 8x

-x² -8x + 105 = 0

(-x + 7)(x + 15) = 0

-x + 7 = 0 or x + 15 = 0

x = 7 or x = -15

The width is 7 meters while the length is 8 meters.

Checking:

A = lw

105 = (x)(x+8)

105 = (7)(7+8)

105 = (7)(15)

105 = 105

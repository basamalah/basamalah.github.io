ANO ANG PRODUKSIYON

• Ang produksiyon ay ang paglikha ng mga produkto at serbisyo.

• Ito ay may input at output

• Ang input ay ang mga ginamit sa paggawa ng isang produkto

• At output ay ang kinalabasang finished products.

26. ano ang kailangan sa paggawa ng sapatos?


Ang mga pangunahing kakailanganin sa paggawa ng sapatos ay ang leather (balat ng hayop) o anumang materyal na gagamitin bilang sapatos (pwede ring tela), pandikit, padding, karayom, sinulid, pangkulay. Kakailanganin mo rin ang martilto, pampukpok, pako o tacks at pangliha.

29. Kasangkapan sa paggawa ng sapatos?​


Materyal para sa iyong sapatos (mas mabuti katad)

- Ang isang pares ng sapatos ay tumatagal (ang mga ito ay maaaring bumili ng pangalawang kamay sa ebay, bago sa Amazon- ang estilo ng huling ay matukoy ang laki at estilo ng sapatos)

- Malagkit (mas mabuti neoprene latagan ng simento o makipag-ugnay malagkit)

- Insole materyal (estilo ng selulusa o katad)

- Single materyal (katad / EVA foam / goma)

- Hair dryer / heat gun

- Mga tool sa paggawa ng sapatos (kutsilyo, pangmatagalang pincers, martilyo, tacks at iba pa) tingnan ang aking itinuturo na "Pagsubok ng Pattern ng Sapatos" para sa karagdagang impormasyon.

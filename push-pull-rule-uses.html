13. How to measure objects using a push and pull rule?


Answer:

PULL / PUSH RULE USES – Definition of pull /push rule, an explanation of the uses of this tool, and its difference between a ruler.

A type of tool used to measure long pieces of stock is the pull/ push rule. Among its kind are: zigzag rule (a folded measuring tool), ruler (to measure short pieces of stock), caliper (for measuring diameter), try square (to check flatness and squareness), measuring stick (a 1-meter measuring device), divider (for dividing distances), compass (for arcs and circles), and protractor (for laying out angles).

6. The area of a carpet is 144sq m. If its length is 2meters less and its width is5 meters more, the carpetwill be square. Find thedimensions of the carpet.​


Answer:

width = 9m; length = 16m

Step-by-step explanation:

First dimension of carpet:

let x be the width

    y be the length

area = xy = 144

Second dimension of carpet: (square)

area = (x + 5)(y - 2) = 144

x + 5 = y - 2

x = y - 7

y (y - 7) = 144

y^2 - 7y - 144 = 0

(y + 9)(y - 16) = 0

y = 16 (-9m is not possible)

x = 16 - 7 = 9

16 * 9 = 144

width = 9m; length = 16m

Answer:

width = 9

length = 16

Step-by-step explanation:

length = L, width = W

L x W = 144                        (because area = length x width)

L - 2 = W + 5                       (because the sides of a square are equal)

L = W+5-2

L = W+7

(W+7)(W) = 144                     (plug in L = W+7)

W^2 + 7W = 144

W^2 + 7W - 144 = 0             (quadratic equation)

(W-9)(W+16) = 0                    (factor)

W=9, or W=-16

Since it cant be negative the answer is W = 9.

Solve for length.

length x 9 = 144

length = 144/9

length = 16

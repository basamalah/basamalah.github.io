Image 1

Charmaine has a circular garden divided into five equal sections. What is the length of the arc of each part if the radius of the garden is 15 m?

360 / 5 = 72°

[tex]\frac{A}{360^{o} } = \frac{L}{2\pi r} = \frac{72^{o} }{360^{o} } = \frac{L}{2\pi (15)} \frac{1}{5} = \frac{L}{30\pi } \frac{30\pi }{5} = L[/tex]

L = 18.84m

Image 2

The piza has a radius of 20 cm. What is the area of the piza sector if the arc of the sliced part of the piza measures 60°?

[tex]\frac{measure of the arc}{360^{o} } = \frac{60^{o} }{30^{o} } = \frac{1}{6}[/tex]

[tex]A = \pi r^{2} A = \pi (20)^{2} A = 400\pi cm^{2}[/tex]

[tex]\frac{1}{6} x (400\pi cm^{2} ) = \frac{400\pi }{6} cm^{2} = \frac{200\pi }{3} cm^{2}[/tex]

= 209.44 cm²

A quadratic equation is an equation (multiple terms) whose highest power is to the degree of two. Usually quadratic equations are often found on curves and curves. The quadratic equation formula is as follows:

ax²+ bx + c= 0

Information:

a, b, and c = real numbers

a, b, and c = constant

x= variable

The quadratic equation in x is an equation of the form ax² + bx + c = 0, where a, b, and c are real numbers and a 0. Values of x that satisfy the equation ax² + bx + c = 0 are called the roots of the quadratic equation.

In relation to the values of a, b, c, there are several names of quadratic equations, including:

a. If a = 1, then the equation becomes x² + bx + c = 0 and an equation like this is called an ordinary quadratic equation.

b. If b = 0, then the equation becomes ax² + c = 0 and such an equation is called a perfect quadratic equation.

c. If c = 0, then the equation becomes ax² + bx = 0 and such an equation is called an incomplete quadratic equation.

d. If a, b, and c are real numbers, then ax² + bx + c = 0 is called a real quadratic equation.

e. If a, b, and c are rational numbers, then ax²+ bx + c = 0 is called a rational quadratic equation.

The characteristics of the quadratic equation are:

An equation, the highest power of the variable is 2 and the lowest power is 0, the coefficient of the variable is a real number, the coefficient of the variable to the power of 2 is not equal to zero, the coefficient of the variable to the power of 1 and 0 can be 0.

How to solve a quadratic equation

As for how to solve a quadratic equation, it can be done in 3 ways, namely factorization, completing the quadratic, and using the abc quadratic equation formula.

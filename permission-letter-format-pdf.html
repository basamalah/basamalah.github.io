23. 5. Why do you think JPEG, PNG, and PDF are printable file formats yet these can still be uploadedonline? Defend your answer.​


Answer:

JPEG, PNG, and PDF are printable file formats because they are designed to produce high-quality images that can be easily printed. They are widely used in graphic design, printing, and publishing industries. JPEG (Joint Photographic Experts Group) is a lossy compression format that is suitable for photographs and complex images. PNG (Portable Network Graphics) is a lossless compression format that supports transparency and is suitable for images with text or sharp lines. PDF (Portable Document Format) is a versatile file format that can contain both text and images, and is commonly used for documents, brochures, and forms.

Although these file formats are designed for printing, they can still be uploaded online because they are compatible with most web browsers and devices. Many websites and online platforms support these file formats, allowing users to upload and share their documents and images online. For example, PDF files can be uploaded to online forms, JPEG and PNG files can be shared on social media or stored in the cloud. Additionally, these file formats are often smaller in size compared to other printable file formats, which makes them easier to upload and download online. Therefore, while these file formats are primarily used for printing, they can also be used for online purposes because they are widely accepted and supported.

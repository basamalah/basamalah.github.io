1. a. If x/y = 10 and dx/dt = -5, find dy/dt b. If [tex]\sqrt{x}[/tex] + [tex]\sqrt{y}[/tex] = 5 and dy/dt = 3, find dx/dt when x = 1 c. If 5x + 3xy = 4 and dy/dt = 3, find dx/dt when x = 2


Solution:

a)

Taking the derivative on both sides,

[tex]x=10y\\[/tex]

[tex]\dfrac{d}{dt}(x)= \dfrac{d}{dt}(10y)\\\\\dfrac{dx}{dt}= 10\dfrac{dy}{dt}[/tex]

Solving for the value of dy/dt,

[tex]\dfrac{dy}{dt}=\dfrac{1}{10}\dfrac{dx}{dt} =\dfrac{1}{10}(-5) \\\\\dfrac{dy}{dt}=-\dfrac{1}{2} \quad\leftarrow\text{ ANSWER}[/tex]

b)

Solving for the value of y,

[tex]\sqrt{x} +\sqrt{y} =5\\y=(5-\sqrt{x} )^2=(5-\sqrt{1} )^2\\y=16[/tex]

Taking the derivative on both sides,

[tex]\sqrt{x} +\sqrt{y} =5\\\\\dfrac{d}{dt}(x^{1/2} +y^{1/2}) =\dfrac{d}{dt}(5)\\\\\dfrac{1}{2} x^{-1/2}\dfrac{dx}{dt}+\dfrac{1}{2}y^{-1/2}\dfrac{dy}{dt}=0[/tex]

Solving for the value of dx/dt,

[tex]\dfrac{dx}{dt}= -x^{3/2}y^{-1/2}\dfrac{dy}{dt}=-(1)^{3/2}(16)^{-1/2}(3)\\\\\dfrac{dx}{dt}=-\dfrac{3}{4}\quad\leftarrow\text{ANSWER}[/tex]

c)

Solving for the value of y,

[tex]5x+3xy=4\\\\y=\dfrac{4-5x}{3x}= \dfrac{4-5(2)}{3(2)}\\\\y=-1[/tex]

Taking the derivative on both sides,

[tex]\dfrac{d}{dt} (5x+3xy)=\dfrac{d}{dt}( 4)\\\\5\dfrac{dx}{dt}+ 3\left( x\dfrac{dy}{dt}+y\dfrac{dx}{dt} \right)=0[/tex]

Solving for the value of dx/dt,

[tex]\dfrac{dx}{dt}=\dfrac{4-3x\dfrac{dy}{dt} }{5+3y} =\dfrac{4-(3)(2)(3)}{5+3(-1)} \\\\\dfrac{dx}{dt}=-7\quad\leftarrow\text{ANSWER}[/tex]
